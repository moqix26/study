## 13. 与后端 Java 的联调对照表

| 前端（Vue） | 后端（Java） | 联调注意 |
|-------------|--------------|----------|
| Axios GET | `@GetMapping` | query 参数名一致 |
| Axios POST JSON | `@RequestBody DTO` | Content-Type: application/json |
| Authorization Header | JWT 拦截器 | Bearer 前缀 |
| `code === 0` 成功 | `Result<T>` 统一返回 | 前后端约定 code |
| `/api/products` | `@RequestMapping("/api")` | 前缀统一 |
| 401 跳登录 | 拦截器返回 401 | 双端处理 |
| Vite proxy `/api` | 8080 端口 | 仅开发 |
| Nginx 反代 `/api` | jar 8080 | 生产 |
| 分页 pageNum/pageSize | PageHelper | 字段名对齐 |
| 表单 username/password | LoginDTO | 校验注解 @Valid |
| 订单 items 数组 | CreateOrderDTO List | JSON 结构一致 |
| CORS 报错 | CorsConfig | 生产同域可免 |

**并行学习对齐**：Vue 08 ↔ 后端 04～05；Vue 10 ↔ 后端 09；Vue 11 ↔ 后端 10。

## 14. HTTP 与接口约定速查

| 状态码 | 含义 | 前端处理 |
|--------|------|----------|
| 200 | OK | 正常（仍看 body.code） |
| 400 | 参数错误 | ElMessage message |
| 401 | 未认证 | 清 token 跳登录 |
| 403 | 无权限 | 提示 |
| 404 | 资源不存在 | 提示或 Empty |
| 500 | 服务器错误 | 提示 + 日志 |

| Method | 用途 | shop-vue 例子 |
|--------|------|---------------|
| GET | 查询 | 商品列表、详情 |
| POST | 创建 | 登录、注册、下单 |
| PUT | 全量更新 | 修改资料（可选） |
| PATCH | 部分更新 | 可选 |
| DELETE | 删除 | 删购物车项（若服务端） |
