## 12. Vue ↔ React 对照总表

| 维度 | Vue 3 | React 18 |
|------|-------|----------|
| 单文件 | `.vue` SFC | `.jsx` / `.tsx` |
| 模板 | `<template>` | JSX |
| 样式 scoped | `<style scoped>` | CSS Modules / styled |
| 响应式 | ref / reactive 自动追踪 | useState 显式更新 |
| 派生数据 | computed | useMemo |
| 副作用监听 | watch / watchEffect | useEffect |
| 双向绑定 | v-model | 受控 value + onChange |
| 父子通信 | props / emit | props / 回调 |
| 跨层通信 | provide / inject | Context |
| 插槽 | slot / 作用域插槽 | children / render props |
| 缓存组件 | KeepAlive | 无内置，组合方案 |
| 传送门 | Teleport | createPortal |
| 异步组件 | defineAsyncComponent | React.lazy |
| 列表 key | :key | key |
| 条件渲染 | v-if / v-show | && / 三元 / 变量 |
| 列表渲染 | v-for | map |
| 路由 | Vue Router | React Router v6 |
| 状态管理 | Pinia | Zustand / Redux |
| UI 库 | Element Plus | Ant Design |
| 构建 | Vite | Vite / CRA |
| 环境变量 | VITE_ | VITE_ |
| 性能优化 | 自动 + v-memo | memo / useMemo 手动 |
| 自定义指令 | directive | 无，用 Hook/组件 |
| 表单校验 | el-form rules | antd Form rules |
| SSR 框架 | Nuxt | Next.js |
| 学习曲线 | 相对平缓 | JSX + 生态选型 |

### 12.1 语法速查对照

| 场景 | Vue | React |
|------|-----|-------|
| 计数器 | `count++`（ref 自动） | `setCount(c => c + 1)` |
| 挂载请求 | `onMounted` | `useEffect(fn, [])` |
| 监听路由 id | `watch(() => route.params.id)` | `useEffect(fn, [id])` |
| 全局用户 | `useUserStore()` | `useUserStore(s => s.token)` |
| 路由跳转 | `router.push('/cart')` | `navigate('/cart')` |
| 懒加载页 | `() => import('./X.vue')` | `lazy(() => import('./X'))` |

### 12.2 项目模块对照（shop-vue ↔ shop-react）

| 模块 | shop-vue | shop-react |
|------|----------|------------|
| 入口 | main.js + createApp | main.jsx + createRoot |

## 15. HTTP 与接口约定速查

| 状态码 | 含义 | 前端处理 |
|--------|------|----------|
| 200 | OK | 正常（仍看 body.code） |
| 400 | 参数错误 | message 提示 |
| 401 | 未认证 | 清 token 跳登录 |
| 403 | 无权限 | 提示 |
| 404 | 资源不存在 | Empty 组件 |
| 500 | 服务器错误 | 提示 + 日志 |

| Method | 用途 | shop-react 例子 |
|--------|------|-----------------|
| GET | 查询 | 商品列表、详情 |
| POST | 创建 | 登录、注册、下单 |
| PUT | 全量更新 | 修改资料（可选） |
| PATCH | 部分更新 | 可选 |
| DELETE | 删除 | 删购物车项（若服务端） |
