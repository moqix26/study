| ul / ol / li | 01 | 导航、列表用列表标签 | 无序/有序列表，每项一个 li | ⬜ |
| a href / target | 01 | 内外链、锚点 | 链接；target=_blank 新窗口打开 | ⬜ |
| img src / alt | 01 | 必写 alt | 图片地址；alt 是加载失败时的文字 | ⬜ |
| 相对路径 / 绝对路径 | 01 | 能画目录结构 | ./ 同级 ../ 上一级；/ 从站点根开始 | ⬜ |

| id / class | 01 | id 唯一，class 可复用 | id 身份证号；class 班级可多人 | ⬜ |
| form / input / button | 02 | 登录注册能写 | 表单收集用户输入 | ⬜ |
| label / textarea / select | 02 | 表单可访问性 | 标签关联输入框；多行文本；下拉 | ⬜ |
| table 基础 | 02 | 知道语义，布局优先 CSS | 表格数据用；页面布局别靠 table | ⬜ |
| video / audio | 02 | 知道标签即可 | 网页里放视频音频 | ⬜ |
| header / nav / main / footer | 02 | 语义化布局 | 页头/导航/主内容/页脚，SEO 友好 | ⬜ |
| 无障碍 alt / label | 02 | 基础意识 | 读屏软件能读；人人能用你的页 | ⬜ |

3. 动态 li；父 ul 绑一次；省内存。
4. GET 查询参数在 URL；POST 常在 body 提交。
5. add 暂存；commit 提交快照。
6. 浏览器拦跨域；Vite proxy 或后端 CORS。
7. `<!DOCTYPE html><html><head><meta charset=UTF-8>...`
8. `display:flex; justify-content:center; align-items:center;`

## 19. 费曼检验

3 分钟解释「学原生前端为什么要先 HTML 再 CSS 再 JS」：

1. **HTML** 是骨架——没有结构谈不上样式和行为。
2. **CSS** 是装修——在骨架上美化。
3. **JS** 是电路——让页面能交互、能请求数据。

## 21. FAQ（本总表）

**Q1：第一优先级（§7）没学完能看 Vue 吗？**  
建议至少完成 HTML+CSS Flex + JS DOM 再开 Vue 01。

**Q2：⬜ 太多怎么办？**  
按 §7 优先级，先攻第一优先级到 🔶。

**Q3：14 章和 13 章区别？**  
13 讲怎么答；14 讲有没有漏。

**Q4：要不要全 ✅ 再面试？**  
第一优先级 🔶 以上 + 13 章场景题 + 1 个项目 demo 即可投初级岗。

**Q5：MDN 和本资料怎么配合？**  
本资料是路线和练习；MDN 查细节语法。

**Q6：Flex 和 Grid 先学哪个？**  
先 Flex（一维导航/列表），再 Grid（二维页面骨架）。

**Q7：fetch 和 Axios 关系？**  
fetch 是浏览器原生；Axios 是封装库，Vue 项目常用 Axios。

**Q8：14 周时间线（§12）太紧？**  
可拉到 20 周；第一优先级不能跳。

**Q9：CodePen 练什么？**  
看别人布局动画；改 small demo 练手。

**Q10：零基础解释列要背吗？**  
理解为主；面试用自己的话讲。
