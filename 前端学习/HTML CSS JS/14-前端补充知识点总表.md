## 12. 学习路径时间线（参考）

```
第 1 周：HTML 基础（01）+ 个人介绍页
第 2 周：HTML 表单、语义化（02）+ 登录/注册页 HTML
第 3 周：CSS 基础、选择器（03）+ 个人卡片
第 4 周：CSS 盒模型、定位（04）+ 商品卡片 + 弹窗
第 5 周：CSS Flex（05 前半）+ 导航栏 + 卡片列表
第 6 周：CSS Grid、响应式（05 后半）+ 整页布局
第 7 周：JS 基础语法（06）+ Console 练习
第 8 周：JS 函数、数组、对象（07）+ 购物车计算
第 9 周：DOM 操作与事件（08）+ Tab 切换 + 弹窗
第 10 周：异步、fetch（09）+ 接口列表页
第 11 周：HTTP、浏览器（10）+ Network 面板分析
第 12 周：工程化、Git（11）+ 整理练习仓库
第 13 周：综合实战（12）+ Todo + 轮播 + 个人主页
第 14 周：面试专题 + 查漏补缺（13-14）
```

**每天 1~2 小时**，14 周可完成全部内容。节奏可根据自身情况调整。

## 14. 常见前端缩写速查

| 缩写 | 全称 | 一句话 |
|------|------|--------|
| BFC | Block Formatting Context | 块级格式化上下文 |
| TDZ | Temporal Dead Zone | let/const 的暂时性死区 |
| IIFE | Immediately Invoked Function Expression | 立即执行函数表达式 |
| SPA | Single Page Application | 单页应用（不刷新页面） |
| MPA | Multi Page Application | 多页应用（传统跳转） |
| CSR | Client-Side Rendering | 客户端渲染（React/Vue 默认） |
| SSR | Server-Side Rendering | 服务端渲染（Next.js/Nuxt） |
| SSG | Static Site Generation | 静态站点生成 |
| JAMstack | JavaScript + APIs + Markup | 现代架构模式 |
| PWA | Progressive Web App | 渐进式 Web 应用（可离线） |
| AJAX | Asynchronous JavaScript And XML | 异步 JS 请求（现在主要用 fetch） |
| REST | Representational State Transfer | RESTful API 风格 |
| CRUD | Create Read Update Delete | 增删改查 |
| DRY | Don't Repeat Yourself | 不要重复自己（代码复用） |
| KISS | Keep It Simple, Stupid | 保持简单 |
