## 0. 读前导读（零基础也能跟上）

> **读者假设**：会基本电脑操作（新建文件夹、保存文件、用浏览器打开网页），**不一定**会编程。本文件是整个 `HTML CSS JS` 系列的「导航图」，不教你写代码，但会告诉你**按什么顺序学、每天练什么、和暑假全栈项目怎么对齐**。

### 0.1 用一句话弄懂本系列

**一句话**：前端三件套 = **HTML 搭骨架、CSS 装修、JavaScript 装电器**；按 01→14 顺序学，每天写代码，六周后你能独立做静态页 + 简单交互，再无缝接 [Vue 3](../Vue/00-学习路线图与说明.md) 和 [Java 后端](../../后端学习/Java/00-学习路线图与说明.md)。

**生活类比——盖房子**：

| 技术 | 房子类比 | 你最先要会的事 |
|------|----------|----------------|
| **HTML** | 钢筋水泥骨架、房间隔断 | 知道哪里是门、哪里是窗（标签与结构） |
| **CSS** | 刷墙、家具摆放、灯光 | 让导航横着排、卡片有间距 |
| **JavaScript** | 开关、门禁、自动窗帘 | 点按钮换 Tab、提交表单前校验 |
| **浏览器 DevTools** | 验房工具 | F12 看结构、看报错、看接口 |

**为什么重要**：简历项目 [todo.md](../../todo.md) 里的 **notehub-fullstack** 最终是 Vue 单页，但 Vue 模板里写的仍是 HTML 标签；HTML/CSS/JS 不扎实，学 Vue 只会「复制 AI 代码却改不动」。

### 0.9 零基础常见误解

| 误解 | 正确理解 |
|------|----------|
| 前端 = 只会 Vue | Vue 是工具；底层仍是 HTML/CSS/JS |
| 标签要全背下来 | 理解语义 + 多写；细节查 [MDN](https://developer.mozilla.org/zh-CN/) |
| CSS 写了一定生效 | 选择器、优先级、路径都可能让样式「消失」 |
| 先学框架最快 | 框架在地基不稳时反而更慢、更焦虑 |
| `div` 万能 | 有语义标签时优先 `nav`/`main`/`article` 等 |
| JS 和 Java 是一回事 | 名字像，语法和运行环境完全不同 |
| 看懂视频 = 会写 | 必须自己敲、自己改错 |
| AI 能代替基础 | AI 生成代码你要能改结构、查 bug |

## 3. 每份文档怎么学（推荐四步法）

### 第一步：通读 + 画结构图

不要一上来就抄代码。先问自己：

1. 这一章解决什么问题？
2. 有哪些核心概念？

- 新建文件夹，例如 `practice/01-html/`

- 用浏览器打开，观察效果
- 故意改错一两处，看浏览器和控制台有什么反应

每份文档都有「练习建议」和「学完标准」。  
**以「学完标准」为过关线**，达不到就回头重看对应小节。

### 第四步：隔天复习 + 用自己的话复述

- 合上书（关文档），用自己的话讲给空气听
- 或写 5 条学习笔记到本文件末尾的「我的笔记区」

## 8. 常见问题 FAQ

### Q1：中文乱码怎么办？

1. HTML 里要有 `<meta charset="UTF-8" />`
2. 文件保存为 UTF-8
3. 编辑器右下角确认编码

### Q2：样式写了为什么不生效？

按这个顺序查：

1. 选择器是否选中了元素？（Elements 面板点选看）
2. 是否被其他样式覆盖？（Styles 面板看删除线）
3. 属性名是否拼错？（`background-color` 不是 `backgroundColor`，除非行内 style）
4. CSS 文件路径是否正确？

### Q3：JavaScript 报错 `Cannot read properties of null`？

通常是 `document.querySelector` 没找到元素。检查：

1. 选择器是否正确
2. `<script>` 是否在 DOM 元素之后，或是否包了 `DOMContentLoaded`

### Q4：需要背下来吗？

**不要死背**。理解 + 多写几遍，自然就记住了。标签、属性、API 随时查 MDN。

### Q5：要不要学框架（React/Vue）？

**先把本文件夹 01-12 过完再学框架。**  
框架是「更高效地写 HTML/CSS/JS」，地基不牢会很痛苦。

学完 HTML/CSS/JS 后，建议按下面顺序扩展（详见 [修改规范](../../修改规范.md)）：

1. **TypeScript**（[TypeScript 00](../TypeScript/00-学习路线图与说明.md)）—— 与框架并行，联调前完成 01～06
2. **Git**（[Git 00](../Git/00-学习路线图与说明.md)）—— 与 Vue/React 01 并行，管理 shop 项目代码
3. **计算机网络**（[计网 00](../计算机网络/00-学习路线图与说明.md)）—— HTML 10 后、框架 08 联调前，系统学 TCP/HTTP/CORS
4. **框架二选一**：
   - [Vue 3](../Vue/00-学习路线图与说明.md)（国内岗位多，Element Plus，Pinia）
   - [React](../React/00-学习路线图与说明.md)（生态最大，Ant Design，Zustand）

两套路由结构平行，均可与后端联调（**Web 业务选 Java 或 Python 一条主线**）：

- [Spring Boot / Java 后端](../../后端学习/Java/00-学习路线图与说明.md)
- [FastAPI / Python 后端](../../后端学习/Python/00-学习路线图与说明.md)

系统编程、算法面试、高性能、**Qt 桌面**见 [C++ 学习路线](../../后端学习/C++/00-学习路线图与说明.md)（非 Web CRUD 主线；**2026 默认可作主线**，见 [todo.md](../../todo.md) 与 [C++ 16 分轨](../../后端学习/C++/16-必学技术栈分轨与扩展专题.md)）。

**框架选一条主线学透即可**。

### Q7：`practice/` 和 `examples/` 有什么区别？

- **`examples/`**：仓库自带、与文档同步的演示文件，**只读参考**，用浏览器打开观察效果。
- **`practice/`**：你自己建的练习目录，可以随意改、改坏再重来；建议纳入 [Git](../Git/01-Git入门与安装配置.md)。

### Q8：学完 01 章就能学 Vue 吗？

**不够。** 至少完成 **01～05（HTML+CSS）+ 06～07（JS 基础）** 再开 [Vue 01](../Vue/01-Vue入门与环境搭建.md)。若暑假赶进度，可 03～05 略读但 **01 与 06 不能跳**。

### Q9：Mac 和 Windows 学前端有区别吗？

核心无区别：都是写 UTF-8 文件 + 浏览器打开。差异在终端命令（Mac 用 bash/zsh，Windows 用 PowerShell）和路径分隔符（`/` vs `\`）；HTML 里路径统一用 **`/`** 即可。

### Q10：需要买课吗？

本仓库 + [MDN 中文](https://developer.mozilla.org/zh-CN/) + [freeCodeCamp](https://www.freecodecamp.org/chinese/) 足够系统自学。付费课适合需要作业批改的人，不是必需品。

### Q11：英文不好能学吗？

术语首次出现本资料会中英对照；查 MDN 时用中文站；代码关键字（`function`、`const`）背发音即可，文档可以中英混读。

### Q12：怎么判断一章「真的学会了」？

1. 章末 **学完标准** 能勾选 ≥ 80%。
2. **闭卷自测** ≥ 7/10（本系列各章标准）。
3. **费曼检验**：能 3 分钟讲给朋友听且对方能复述大意。
4. **隔天** 不看文档重写核心示例。

### Q13：和 ACM/C++ 背景怎么配合？

算法功底让你学 JS 循环、数组更快；但前端还考 **DOM、CSS 布局、浏览器行为**，不能只用刷题思维。建议每天 **一半写页面、一半刷题**（见 [todo.md](../../todo.md) 每日固定动作）。

### Q14：Live Server 必须装吗？

不必须。双击 `.html` 就能学 01～08；装 Live Server 后**保存自动刷新**，学 CSS/JS 时更省事（11 章会再提）。

## 11. 闭卷自测

合上书、关掉文档，尝试回答（共 10 题：6 概念 + 2 动手 + 2 综合）。

1. **概念** 用一句话说明 HTML、CSS、JavaScript 各负责什么？
2. **概念** 为什么建议按 01→14 顺序学，而不是直接学 Vue？
3. **概念** UTF-8 乱码时你要检查哪三件事？
4. **概念** `practice/` 和 `examples/` 目录各是什么用途？
5. **概念** 学完 HTML 10 和计网 06 对 Vue 联调有什么帮助？
6. **概念** 「四步法」的第三步是什么？过关线是什么？
7. **动手** 新建 `practice/00-test/readme.txt`，写一句你的学习目标（证明你会建目录）。
8. **动手** 打开浏览器 F12，说出 Elements / Console / Network 三个面板各看什么。
9. **综合** 对照 [todo.md](../../todo.md)，暑假第 3 周你应该主攻哪条技术线？需要哪几章 HTML/JS 打底？
10. **综合** 你想走 Vue + Java 全栈：列出前端、后端、工程化各一个对应的学习路径文件名（00 章即可）。

### 自测参考答案

1. HTML 结构与内容；CSS 样式与布局；JS 交互与逻辑。
2. 框架是上层工具；没有 HTML/CSS/JS 地基，改模板、排错、布局都会很痛苦。
3. HTML 有 `<meta charset="UTF-8" />`；文件保存为 UTF-8；编辑器右下角编码为 UTF-8。
4. `examples/` 仓库演示只读；`practice/` 自己练习可随意修改。
5. 懂 HTTP 状态码、请求方法、CORS，才能和后端联调、用 Network 排错。
6. 第三步是做章末练习并以「学完标准」过关；达不到就重看对应小节。
7. （无标准答案）能创建文件并写入即可。
8. Elements 看 DOM 结构与样式；Console 看 JS 输出与报错；Network 看请求与响应（09、10 章常用）。
9. 第 3 周主攻 **Vue 01～09 + Axios 联调**；打底需要 HTML 01～05、JS 06～09、10 HTTP 概念。
10. 示例：前端 [Vue 00](../Vue/00-学习路线图与说明.md)；后端 [Java 00](../../后端学习/Java/00-学习路线图与说明.md)；工程化 [Git 00](../Git/00-学习路线图与说明.md)。
