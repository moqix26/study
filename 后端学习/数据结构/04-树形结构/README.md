# 树形结构模块导航

树模块按“遍历 → 有序搜索 → 自平衡 → 专用树形结构”递进。精读时要抓住每种树的**不变量**，因为插入、删除和旋转都只是为了恢复不变量。

| 顺序 | 文档 | 核心不变量或能力 |
|---:|---|---|
| 1 | [二叉树与遍历](01-二叉树与遍历.md) | 每节点最多两个孩子；DFS/BFS 访问顺序 |
| 2 | [二叉搜索树](02-二叉搜索树.md) | 左子树 `<` 根 `<` 右子树 |
| 3 | [AVL 树](03-AVL树.md) | 任意节点左右高度差绝对值不超过 1 |
| 4 | [红黑树](04-红黑树.md) | 颜色规则保证近似平衡 |
| 5 | [堆](05-堆.md) | 完全二叉树 + 父子偏序 |
| 6 | [并查集](06-并查集.md) | 每个集合由代表元管理，路径压缩与按秩合并 |
| 7 | [Trie](07-Trie字典树.md) | 从根到节点的路径表示字符串前缀 |

## 结构对比

| 结构 | 是否全局有序 | 主要用途 | 典型复杂度 |
|---|---|---|---|
| 普通二叉树 | 否 | 层次关系、递归分解 | 遍历 `O(n)` |
| BST | 是 | 动态查询与范围遍历 | 平均 `O(log n)`，最坏 `O(n)` |
| AVL | 是 | 查询密集、严格平衡 | `O(log n)` |
| 红黑树 | 是 | 查询与更新均衡 | `O(log n)` |
| 堆 | 仅父子偏序 | 动态最值、TopK | 插入/删顶 `O(log n)` |
| 并查集 | 不适用 | 动态连通性 | 均摊 `O(α(n))` |
| Trie | 字符路径有序 | 前缀与词典匹配 | `O(L)` |

建议至少手动画一次 AVL 的 LL/LR/RR/RL 和红黑树插入修复，再运行三语言代码。
