## 选型提示

| 条件 | 常见选择 |
|---|---|
| 小规模或近乎有序 | 插入排序 |
| 需要稳定且接受 `O(n)` 辅助空间 | 归并排序 |
| 通用内存排序、平均性能优秀 | 快速排序 |
| 要求原地且最坏 `O(n log n)` | 堆排序 |
| 整数值域较小 | 计数排序 |
| 数据近似均匀分布 | 桶排序 |
| 固定位数整数或字符串 | 基数排序 |

除了复杂度，还要说明稳定性、是否原地、输入分布与最坏情况。
