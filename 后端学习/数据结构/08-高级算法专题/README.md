# 高级算法思想模块导航

这些专题不是新的容器，而是组织搜索过程、缩小枚举范围或复用子问题答案的方法。

| 顺序 | 文档 | 识别信号 |
|---:|---|---|
| 1 | [滑动窗口与双指针](01-滑动窗口与双指针.md) | 连续区间、单调移动、去重或配对 |
| 2 | [回溯](02-回溯.md) | 枚举选择、撤销选择、需要剪枝 |
| 3 | [分治](03-分治.md) | 子问题相对独立，可合并结果 |
| 4 | [动态规划基础](04-动态规划基础.md) | 重叠子问题、最优子结构、状态转移 |

## 思想对比

| 方法 | 是否枚举 | 是否复用答案 | 关键设计 |
|---|---|---|---|
| 双指针/滑窗 | 缩小枚举 | 通常否 | 指针单调性、窗口不变量 |
| 回溯 | 是 | 通常否 | 选择列表、路径、终止条件、剪枝 |
| 分治 | 分成子问题 | 通常不缓存 | 划分、递归、合并 |
| 动态规划 | 隐式枚举状态 | 是 | 状态、转移、初始化、顺序 |

看到题目时先写朴素解法，再寻找单调性、重复计算或可剪枝条件，避免直接套模板。
