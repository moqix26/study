# 线性结构模块导航

线性结构中的元素按“一前一后”组织，是理解指针、容器、缓存局部性和后续树图结构的基础。

| 顺序 | 文档 | 重点 |
|---:|---|---|
| 1 | [数组](01-数组.md) | 连续内存、随机访问、扩容、插入删除 |
| 2 | [链表](02-链表.md) | 单链、双链、循环链、指针更新顺序 |
| 3 | [栈与队列](03-栈与队列.md) | LIFO、FIFO、双端队列、循环队列 |

推荐对比：

| 需求 | 首选 |
|---|---|
| 高频按下标访问、顺序扫描 | 数组 |
| 已知节点位置后频繁插入删除 | 链表 |
| 最近进入的数据先处理 | 栈 |
| 按到达顺序处理 | 队列 |
| 两端都需要入队出队 | 双端队列 |
| 固定容量缓冲区 | 循环队列 |

学完后应能从“内存布局、操作复杂度、边界条件”三个角度解释选型，而不只是背 API。
