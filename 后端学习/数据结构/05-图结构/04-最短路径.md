# 最短路径：Dijkstra 与 Floyd

## 学习目标

- 区分单源最短路径与全源最短路径，理解二者的输出和成本差异。
- 掌握 Dijkstra 的松弛、最小堆优化、路径恢复及非负权前提。
- 掌握 Floyd-Warshall 的动态规划状态、转移顺序及负环检测。
- 能依据有向/无向、稀疏/稠密、是否含负权边和查询次数选择算法。

## 核心原理

### 1. 问题模型与“不可达”

一条路径由首尾相接的边组成，路径长度是所有边权之和。根据查询范围，最短路径可分为：

- **单源**：固定源点 `s`，求 `s` 到每个顶点的最短距离。
- **单对**：只关心 `s` 到 `t`。某些算法可以到达 `t` 后提前停止，但仍必须保证停止条件正确。
- **全源**：求任意 `i` 到任意 `j` 的距离，结果通常是 `V × V` 矩阵。

方向是问题定义的一部分。有向边 `u -> v` 不能反向使用，因此 `distance(u,v)` 与 `distance(v,u)` 可能不同；无向边在邻接表中通常存成两条等权有向记录。

不可达不等于距离 0。`dist[s]=0` 是因为从源点到自身可以走空路径；其他尚不可达顶点应初始化为一个足够大的哨兵 `INF`。`INF` 必须留出加法余量，例如使用整数类型最大值的四分之一，避免 `INF + weight` 溢出后变成负数。

三种语言示例使用以下有向边：

| 边 | 权值 | 边 | 权值 |
|---|---:|---|---:|
| `0->1` | 4 | `0->2` | 1 |
| `2->1` | 2 | `1->3` | 1 |
| `2->3` | 5 | `3->4` | 3 |
| `1->4` | 7 |  |  |

### 2. 松弛：用一条边改进当前答案

`dist[v]` 表示目前已经发现的、从源点到 `v` 的最短路径长度。只要它不是 `INF`，就对应某条真实路径，所以它是实际最短距离的一个上界；算法通过不断降低这个上界逼近最终答案。

检查边 `u -> v`，权值为 `w` 时，若 `u` 已可达，就能把“某条到 `u` 的路径”与这条边拼接，得到到 `v` 的候选路径：

`candidate = dist[u] + w`

若 `candidate < dist[v]`，便执行：

- `dist[v] = candidate`
- `parent[v] = u`

这一步称为**松弛**。名称可以理解为原先对 `v` 的距离估计过紧或过大，现在允许它下降到更好的值。若候选值相等，距离无需更新；是否更新前驱取决于题目是否要求特定的平局路径。

最短路径算法之间的核心差别不是松弛公式，而是“按什么顺序选择顶点、每条边需要松弛多少次”。BFS 按层松弛等权边；Dijkstra 按当前最小距离处理非负权边；Bellman-Ford 多轮扫描所有边；Floyd 则按允许的中间顶点范围更新点对距离。

### 3. Dijkstra 的状态与贪心选择

Dijkstra 适用于所有边权非负的单源最短路。堆优化版本维护：

- `dist[v]`：当前最佳已知距离。
- `parent[v]`：当前最佳路径上 `v` 的前驱。
- 最小堆中的 `(候选距离, 顶点)`：等待展开的候选状态。

初始化 `dist[source]=0`，其余为 `INF`，并把 `(0,source)` 入堆。每次弹出堆中距离最小的状态：

1. 若弹出的距离不等于当前 `dist[u]`，它是旧状态，直接跳过。
2. 否则用 `dist[u]` 松弛 `u` 的全部出边。
3. 每次成功降低 `dist[v]`，把新的 `(dist[v],v)` 再次入堆。

这里没有在堆中执行 decrease-key，而是允许同一顶点存在多个候选。新值更小，会更早弹出；旧值日后通过比较被识别为过期项。这种懒删除是工程实现中最常见的写法。

Dijkstra 的关键不变量是：**当一个与当前 `dist[u]` 相等的最小堆状态被取出时，在非负权前提下，`dist[u]` 已经是最终最短距离。** 后续即使继续发现其他路径，也不可能把它降低。

### 4. Dijkstra 的完整堆轨迹

从源点 0 运行示例图，距离数组按 `[0,1,2,3,4]` 排列：

| 步骤 | 弹出状态 | 有效松弛 | 松弛后的 `dist` | 堆中关键候选 |
|---:|---|---|---|---|
| 初始 | 无 | 无 | `[0,INF,INF,INF,INF]` | `(0,0)` |
| 1 | `(0,0)` | `0->1: 4`，`0->2: 1` | `[0,4,1,INF,INF]` | `(1,2),(4,1)` |
| 2 | `(1,2)` | `2->1: 1+2=3`，`2->3: 1+5=6` | `[0,3,1,6,INF]` | `(3,1),(4,1),(6,3)` |
| 3 | `(3,1)` | `1->3: 3+1=4`，`1->4: 3+7=10` | `[0,3,1,4,10]` | `(4,1旧),(4,3),(6,3旧),(10,4)` |
| 4 | `(4,1)` | 过期，因当前 `dist[1]=3` | 不变 | `(4,3),(6,3旧),(10,4)` |
| 5 | `(4,3)` | `3->4: 4+3=7` | `[0,3,1,4,7]` | `(6,3旧),(7,4),(10,4旧)` |
| 6 | `(6,3)` | 过期，因当前 `dist[3]=4` | 不变 | `(7,4),(10,4旧)` |
| 7 | `(7,4)` | 无出边 | `[0,3,1,4,7]` | `(10,4旧)` |
| 8 | `(10,4)` | 过期，因当前 `dist[4]=7` | 不变 | 空 |

最终最短距离为 `[0,3,1,4,7]`。表中 `(4,1)`、`(6,3)`、`(10,4)` 正是懒删除产生的旧候选；如果省略过期检查，虽然重复松弛不一定立刻算错，但会造成大量无效工作，并使算法行为难以分析。

### 5. 非负权为什么是硬前提

设 `u` 是当前堆中距离最小的有效顶点。假设还存在一条更短但尚未发现的源点到 `u` 路径。在这条路径上，取第一个尚未按最终距离处理的顶点 `x`，它的前驱 `y` 已经处理。处理 `y` 时应已经松弛 `y->x`，而所有剩余边权非负，所以到 `x` 再继续走到 `u` 不可能先超过 `dist[u]` 后又凭负边降回来。这与 `u` 当前最小矛盾。

负权边会破坏“距离只会越走越大”的基础。例如有 `0->A(2)、0->B(5)、B->A(-10)`：A 看起来会先以距离 2 被选中，但后来经过 B 可得到距离 -5。此时“首次取出 A 即最终确定”的结论失效。某些允许顶点反复入堆的实现可能在个别无负环输入上仍碰巧修正结果，但那已不再具备 Dijkstra 的正确性与复杂度保证，不能作为使用负边的依据。

含负权边的单源问题应使用 Bellman-Ford 等允许反复传播改进的算法。若存在从源点可达的负环，沿环多走一次就能让总权继续下降，因此相关终点不存在有限最短距离。

### 6. 路径恢复及多条最短路

每次松弛成功时记录 `parent[v]=u`，表示当前最佳路径到达 `v` 的最后一条边来自 `u`。示例中最终前驱为：

- `parent[2]=0`
- `parent[1]=2`
- `parent[3]=1`
- `parent[4]=3`

恢复 0 到 4 的路径时，从目标反向得到 `4<-3<-1<-2<-0`，再反转为 `0->2->1->3->4`，边权和为 `1+2+1+3=7`。

恢复前必须先检查 `dist[target] != INF`。不可达顶点的前驱通常保持 `-1`，它不代表一条合法路径。有多条等长最短路时，`<` 更新会保留先发现的那条；如果要求字典序最小、边数最少或其他次级目标，需要在状态比较和前驱更新中明确加入规则。

### 7. Floyd-Warshall 的阶段状态

Floyd-Warshall 一次计算所有点对距离。初始矩阵满足：

- `dist[i][i]=0`。
- 直接边 `i->j` 的位置为边权；若有重边，保留最小权。
- 没有直接边的位置为 `INF`。

为了理解三重循环，定义 `D^(k)[i][j]`：从 `i` 到 `j`、且允许作为**中间顶点**的编号只来自 `0..k` 时的最短距离。端点 `i、j` 不受这个编号限制。

第 `k` 轮只需考虑最优路径是否经过顶点 `k`：

- 不经过 `k`：沿用上一阶段的 `dist[i][j]`。
- 经过 `k`：最短路径可拆成 `i -> k` 和 `k -> j`，候选为 `dist[i][k] + dist[k][j]`。

所以转移为：

`dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j])`

这也是 `k` 必须位于最外层的原因：完成整轮 `(i,j)` 更新后，矩阵才统一表示“已允许 0..k 作为中间点”。若把 `i` 或 `j` 放在最外层，更新会混用不同阶段的值，失去这一动态规划不变量。

### 8. Floyd 在示例矩阵上的更新

只列出能体现变化的关键点：

| 允许的新增中间点 | 被改进的点对 | 原距离 | 新路径 | 新距离 |
|---:|---|---:|---|---:|
| 1 | `0->3` | `INF` | `0->1->3 = 4+1` | 5 |
| 1 | `2->3` | 5 | `2->1->3 = 2+1` | 3 |
| 1 | `2->4` | `INF` | `2->1->4 = 2+7` | 9 |
| 2 | `0->1` | 4 | `0->2->1 = 1+2` | 3 |
| 2 | `0->3` | 5 | `0->2->3 = 1+3` | 4 |
| 3 | `0->4` | 10 | `0->3->4 = 4+3` | 7 |
| 3 | `1->4` | 7 | `1->3->4 = 1+3` | 4 |
| 3 | `2->4` | 9 | `2->3->4 = 3+3` | 6 |

这里 `0->2->3` 中的 `2->3=3` 已在允许中间点 1 的阶段得到，正体现了分阶段复用子问题结果。最终矩阵第一行为 `0,3,1,4,7`，与从 0 运行 Dijkstra 的结果一致。

在相加前必须检查 `dist[i][k]` 与 `dist[k][j]` 都不是 `INF`。否则“不可达哨兵 + 权值”没有路径含义，还可能发生整数溢出。

### 9. Floyd 与负环

Floyd 允许负权边，但有限最短路要求相关路径上不能利用负环。算法结束后若某个 `dist[k][k] < 0`，说明存在一条从 `k` 出发又回到 `k` 的负权闭合路径；重复它可以让总距离无限下降。

受影响的不只是 `k` 到自身。对任意点对 `(i,j)`，如果：

- `i` 能到达这个负环，即 `dist[i][k] != INF`；并且
- 从负环能到达 `j`，即 `dist[k][j] != INF`，

那么 `i` 到 `j` 也没有有限最短距离。实际程序若要完整表达结果，应在 Floyd 后再扫描所有负对角线顶点，并标记这些受影响点对，而不是只输出一个“存在负环”的布尔值。

### 10. 算法选择与复杂度来源

| 条件 | 推荐方法 | 选择理由 |
|---|---|---|
| 无权图或所有边等权 | BFS | 按层即可得到最少边数，`O(V+E)` |
| 非负权、单个或少量源点、稀疏图 | 邻接表 + 堆 Dijkstra | 只扫描实际边，约 `O((V+E)log V)` |
| 非负权、稠密图、单源 | 邻接矩阵 Dijkstra | `O(V²)`，可能优于堆的常数和边存储 |
| 顶点数不大、需要所有点对 | Floyd-Warshall | 一次 `O(V³)` 预处理，之后每次距离查询 `O(1)` |
| 含负边的单源 | Bellman-Ford | 能传播负边影响并检测源点可达负环 |
| 稀疏图的多源或全源 | 多次 Dijkstra / Johnson | 可避免 Floyd 对大量不存在边的组合计算 |

堆版 Dijkstra 中，每次成功松弛都会新增一个堆状态，状态数为 `O(E)`，每次堆操作为 `O(log E)`，严格可写 `O((V+E)log E)`；简单图中 `log E=O(log V)`，常写成 `O((V+E)log V)`。Floyd 的三层循环各遍历 `V` 个顶点，时间 `O(V³)`，距离矩阵空间 `O(V²)`。

## 复杂度表

| 算法/实现 | 适用权值 | 时间复杂度 | 额外空间 | 主要用途 |
|---|---|---:|---:|---|
| BFS | 每条边等权/无权 | `O(V+E)` | `O(V)` | 单源最少边数 |
| Dijkstra：邻接表 + 懒删除堆 | 非负权 | `O((V+E) log E)`；简单图可写成 `O((V+E) log V)` | `O(V+E)` | 稀疏图单源 |
| Dijkstra：邻接矩阵 | 非负权 | `O(V²)` | `O(V)`（矩阵本身 `O(V²)`） | 稠密图单源 |
| Floyd-Warshall | 可含负边，不可有相关负环 | `O(V³)` | `O(V²)` | 全源最短路 |

## 适用场景

- 地图导航、网络延迟、费用最小化等非负权单源查询：Dijkstra。
- 小规模城市间距离表、任意两点重复查询、传递闭包变种：Floyd。
- 路径不仅要距离还要节点序列：Dijkstra 记录 `parent`；Floyd 可维护 `next[i][j]`。
- 注意“最短”优化的是路径和，若目标是最少换乘、最大瓶颈、概率最大等，需要调整状态和运算规则。

## C++17 完整示例

```cpp
#include <algorithm>
#include <iostream>
#include <limits>
#include <queue>
#include <string>
#include <utility>
#include <vector>
using namespace std;

const long long INF = numeric_limits<long long>::max() / 4;

struct Edge {
    int from;
    int to;
    int weight;
};

pair<vector<long long>, vector<int>> dijkstra(
    int source, const vector<vector<pair<int, int>>>& adjacency) {
    int n = static_cast<int>(adjacency.size());
    vector<long long> dist(n, INF);
    vector<int> parent(n, -1);
    priority_queue<pair<long long, int>,
                   vector<pair<long long, int>>,
                   greater<pair<long long, int>>> heap;

    dist[source] = 0;
    heap.push({0, source});
    while (!heap.empty()) {
        auto [currentDistance, u] = heap.top();
        heap.pop();
        if (currentDistance != dist[u]) continue;

        for (auto [v, weight] : adjacency[u]) {
            long long candidate = currentDistance + weight;
            if (candidate < dist[v]) {
                dist[v] = candidate;
                parent[v] = u;
                heap.push({candidate, v});
            }
        }
    }
    return {dist, parent};
}

vector<int> restorePath(int source, int target,
                        const vector<long long>& dist,
                        const vector<int>& parent) {
    if (dist[target] == INF) return {};
    vector<int> path;
    for (int current = target; current != -1; current = parent[current]) {
        path.push_back(current);
    }
    reverse(path.begin(), path.end());
    if (path.empty() || path.front() != source) return {};
    return path;
}

vector<vector<long long>> floydWarshall(int n, const vector<Edge>& edges) {
    vector<vector<long long>> dist(n, vector<long long>(n, INF));
    for (int i = 0; i < n; ++i) dist[i][i] = 0;
    for (const Edge& edge : edges) {
        dist[edge.from][edge.to] = min(
            dist[edge.from][edge.to], static_cast<long long>(edge.weight));
    }

    for (int k = 0; k < n; ++k) {
        for (int i = 0; i < n; ++i) {
            if (dist[i][k] == INF) continue;
            for (int j = 0; j < n; ++j) {
                if (dist[k][j] == INF) continue;
                dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j]);
            }
        }
    }
    return dist;
}

int main() {
    const int n = 5;
    vector<Edge> edges = {
        {0, 1, 4}, {0, 2, 1}, {2, 1, 2}, {1, 3, 1},
        {2, 3, 5}, {3, 4, 3}, {1, 4, 7}
    };
    vector<vector<pair<int, int>>> adjacency(n);
    for (const Edge& edge : edges) {
        adjacency[edge.from].push_back({edge.to, edge.weight});
    }

    auto [dist, parent] = dijkstra(0, adjacency);
    cout << "Dijkstra from 0:\n";
    for (int target = 0; target < n; ++target) {
        cout << target << ": distance=" << dist[target] << " path=";
        vector<int> path = restorePath(0, target, dist, parent);
        for (int i = 0; i < static_cast<int>(path.size()); ++i) {
            if (i > 0) cout << "->";
            cout << path[i];
        }
        cout << '\n';
    }

    auto allPairs = floydWarshall(n, edges);
    cout << "Floyd distance matrix:\n";
    for (const auto& row : allPairs) {
        for (int j = 0; j < n; ++j) {
            if (j > 0) cout << ' ';
            if (row[j] == INF) cout << "INF";
            else cout << row[j];
        }
        cout << '\n';
    }
    return 0;
}
```

## Java 17 完整示例

```java
import java.util.ArrayList;
import java.util.Arrays;
import java.util.Collections;
import java.util.Comparator;
import java.util.List;
import java.util.PriorityQueue;

public class Main {
    static final long INF = Long.MAX_VALUE / 4;

    static class Edge {
        final int from;
        final int to;
        final int weight;

        Edge(int from, int to, int weight) {
            this.from = from;
            this.to = to;
            this.weight = weight;
        }
    }

    static class State {
        final int vertex;
        final long distance;

        State(int vertex, long distance) {
            this.vertex = vertex;
            this.distance = distance;
        }
    }

    static class DijkstraResult {
        final long[] dist;
        final int[] parent;

        DijkstraResult(long[] dist, int[] parent) {
            this.dist = dist;
            this.parent = parent;
        }
    }

    static DijkstraResult dijkstra(int source, List<List<Edge>> adjacency) {
        int n = adjacency.size();
        long[] dist = new long[n];
        int[] parent = new int[n];
        Arrays.fill(dist, INF);
        Arrays.fill(parent, -1);
        PriorityQueue<State> heap = new PriorityQueue<>(
            Comparator.comparingLong(state -> state.distance));

        dist[source] = 0;
        heap.offer(new State(source, 0));
        while (!heap.isEmpty()) {
            State current = heap.poll();
            int u = current.vertex;
            if (current.distance != dist[u]) continue;

            for (Edge edge : adjacency.get(u)) {
                long candidate = current.distance + edge.weight;
                if (candidate < dist[edge.to]) {
                    dist[edge.to] = candidate;
                    parent[edge.to] = u;
                    heap.offer(new State(edge.to, candidate));
                }
            }
        }
        return new DijkstraResult(dist, parent);
    }

    static List<Integer> restorePath(int source, int target,
                                     DijkstraResult result) {
        if (result.dist[target] == INF) return Collections.emptyList();
        List<Integer> path = new ArrayList<>();
        for (int current = target; current != -1; current = result.parent[current]) {
            path.add(current);
        }
        Collections.reverse(path);
        if (path.isEmpty() || path.get(0) != source) return Collections.emptyList();
        return path;
    }

    static long[][] floydWarshall(int n, List<Edge> edges) {
        long[][] dist = new long[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(dist[i], INF);
            dist[i][i] = 0;
        }
        for (Edge edge : edges) {
            dist[edge.from][edge.to] = Math.min(
                dist[edge.from][edge.to], edge.weight);
        }

        for (int k = 0; k < n; k++) {
            for (int i = 0; i < n; i++) {
                if (dist[i][k] == INF) continue;
                for (int j = 0; j < n; j++) {
                    if (dist[k][j] == INF) continue;
                    dist[i][j] = Math.min(
                        dist[i][j], dist[i][k] + dist[k][j]);
                }
            }
        }
        return dist;
    }

    public static void main(String[] args) {
        int n = 5;
        List<Edge> edges = Arrays.asList(
            new Edge(0, 1, 4), new Edge(0, 2, 1),
            new Edge(2, 1, 2), new Edge(1, 3, 1),
            new Edge(2, 3, 5), new Edge(3, 4, 3),
            new Edge(1, 4, 7)
        );
        List<List<Edge>> adjacency = new ArrayList<>();
        for (int i = 0; i < n; i++) adjacency.add(new ArrayList<>());
        for (Edge edge : edges) adjacency.get(edge.from).add(edge);

        DijkstraResult result = dijkstra(0, adjacency);
        System.out.println("Dijkstra from 0:");
        for (int target = 0; target < n; target++) {
            System.out.print(target + ": distance=" + result.dist[target] + " path=");
            List<Integer> path = restorePath(0, target, result);
            for (int i = 0; i < path.size(); i++) {
                if (i > 0) System.out.print("->");
                System.out.print(path.get(i));
            }
            System.out.println();
        }

        long[][] allPairs = floydWarshall(n, edges);
        System.out.println("Floyd distance matrix:");
        for (long[] row : allPairs) {
            for (int j = 0; j < n; j++) {
                if (j > 0) System.out.print(" ");
                System.out.print(row[j] == INF ? "INF" : row[j]);
            }
            System.out.println();
        }
    }
}
```

## Go 完整示例

```go
package main

import (
	"container/heap"
	"fmt"
	"math"
)

const INF int64 = math.MaxInt64 / 4

type Edge struct {
	from, to int
	weight   int64
}

type State struct {
	vertex   int
	distance int64
}

type StateHeap []State

func (h StateHeap) Len() int           { return len(h) }
func (h StateHeap) Less(i, j int) bool { return h[i].distance < h[j].distance }
func (h StateHeap) Swap(i, j int)      { h[i], h[j] = h[j], h[i] }
func (h *StateHeap) Push(x any)        { *h = append(*h, x.(State)) }
func (h *StateHeap) Pop() any {
	old := *h
	last := old[len(old)-1]
	*h = old[:len(old)-1]
	return last
}

func dijkstra(source int, adjacency [][]Edge) ([]int64, []int) {
	n := len(adjacency)
	dist := make([]int64, n)
	parent := make([]int, n)
	for i := range dist {
		dist[i] = INF
		parent[i] = -1
	}
	dist[source] = 0
	states := &StateHeap{{vertex: source, distance: 0}}
	heap.Init(states)

	for states.Len() > 0 {
		current := heap.Pop(states).(State)
		u := current.vertex
		if current.distance != dist[u] {
			continue
		}
		for _, edge := range adjacency[u] {
			candidate := current.distance + edge.weight
			if candidate < dist[edge.to] {
				dist[edge.to] = candidate
				parent[edge.to] = u
				heap.Push(states, State{vertex: edge.to, distance: candidate})
			}
		}
	}
	return dist, parent
}

func restorePath(source, target int, dist []int64, parent []int) []int {
	if dist[target] == INF {
		return nil
	}
	path := make([]int, 0)
	for current := target; current != -1; current = parent[current] {
		path = append(path, current)
	}
	for left, right := 0, len(path)-1; left < right; left, right = left+1, right-1 {
		path[left], path[right] = path[right], path[left]
	}
	if len(path) == 0 || path[0] != source {
		return nil
	}
	return path
}

func floydWarshall(n int, edges []Edge) [][]int64 {
	dist := make([][]int64, n)
	for i := 0; i < n; i++ {
		dist[i] = make([]int64, n)
		for j := 0; j < n; j++ {
			dist[i][j] = INF
		}
		dist[i][i] = 0
	}
	for _, edge := range edges {
		if edge.weight < dist[edge.from][edge.to] {
			dist[edge.from][edge.to] = edge.weight
		}
	}
	for k := 0; k < n; k++ {
		for i := 0; i < n; i++ {
			if dist[i][k] == INF {
				continue
			}
			for j := 0; j < n; j++ {
				if dist[k][j] == INF {
					continue
				}
				candidate := dist[i][k] + dist[k][j]
				if candidate < dist[i][j] {
					dist[i][j] = candidate
				}
			}
		}
	}
	return dist
}

func main() {
	n := 5
	edges := []Edge{
		{0, 1, 4}, {0, 2, 1}, {2, 1, 2}, {1, 3, 1},
		{2, 3, 5}, {3, 4, 3}, {1, 4, 7},
	}
	adjacency := make([][]Edge, n)
	for _, edge := range edges {
		adjacency[edge.from] = append(adjacency[edge.from], edge)
	}

	dist, parent := dijkstra(0, adjacency)
	fmt.Println("Dijkstra from 0:")
	for target := 0; target < n; target++ {
		fmt.Printf("%d: distance=%d path=", target, dist[target])
		path := restorePath(0, target, dist, parent)
		for i, vertex := range path {
			if i > 0 {
				fmt.Print("->")
			}
			fmt.Print(vertex)
		}
		fmt.Println()
	}

	allPairs := floydWarshall(n, edges)
	fmt.Println("Floyd distance matrix:")
	for _, row := range allPairs {
		for j, value := range row {
			if j > 0 {
				fmt.Print(" ")
			}
			if value == INF {
				fmt.Print("INF")
			} else {
				fmt.Print(value)
			}
		}
		fmt.Println()
	}
}
```

## 易错点与要点总结

- Dijkstra 的硬前提是所有可用边权非负；“示例刚好算对”不能证明它可用于负权图。
- 无向图添加边时必须存两个方向；有向图不能擅自补反向边。
- 堆优化 Dijkstra 会产生过期状态，弹出后必须与当前 `dist[u]` 比较。
- `INF + weight` 可能溢出；Floyd 转移前应检查两段可达，并使用足够大的整数类型。
- Floyd 的 `k` 必须在最外层。随意交换三重循环顺序会破坏动态规划阶段含义。
- 重边要保留最小权值；矩阵初始化时不能让后出现的更大重边覆盖更小边。
- `dist[i][i] < 0` 表示负环，但还要结合可达性判断哪些点对受该负环影响。
- 路径不可达时要单独输出 `INF`/空路径，不能把未初始化前驱当作合法节点。
- 最短路径树不一定是 MST：前者最小化某个源点到各点的距离，后者最小化整棵树的总边权。
