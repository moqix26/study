## 6. 独立复核中修正的关键问题

- 补足 AVL 删除中子树平衡因子为 0 时的旋转选择、红黑树黑高约定与双黑修复轨迹；
- 明确 Trie 删除实现范围、前缀定位成本与输出全部结果的额外成本；
- 补足 DFS 拓扑排序的进入、完成、反转序列及三色环检测过程；
- 修正 N 皇后在保存棋盘时的 O(n²) 工作空间和结果构造成本；
- 统一桶排序中桶数符号，补充计数、桶、基数排序的值域与整数边界；
- 严格区分哈希表元素数 n 与容量 m：开放寻址扫描整表的最坏时间为 O(m)；
- C++ 哈希示例不再依赖标准未保证的 `std::hash<int>` 具体映射；
- 完善递归空间、均摊复杂度、辅助空间与总空间、并查集 `alpha(n)` 等结论的成立前提。
