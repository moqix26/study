| 复杂度 | 含义 | 典型例子 |
|--------|------|----------|
| O(1) | 与规模无关 | HashMap 平均 get |

## 23. 数组与双指针（Java 模板）

### 两数之和（LeetCode 1 思路）

```java
public int[] twoSum(int[] nums, int target) {
    Map<Integer, Integer> map = new HashMap<>();
    for (int i = 0; i < nums.length; i++) {
        int need = target - nums[i];
        if (map.containsKey(need)) {
            return new int[] { map.get(need), i };
        }
        map.put(nums[i], i);
    }
    return new int[0];
}
```

### 有序数组去重（双指针）

```java
public int removeDuplicates(int[] nums) {
    if (nums.length == 0) return 0;
    int slow = 0;
    for (int fast = 1; fast < nums.length; fast++) {
        if (nums[fast] != nums[slow]) {
            slow++;
            nums[slow] = nums[fast];
        }
    }
    return slow + 1;
}
```

## 24. 链表（Java 模板）

```java
class ListNode {
    int val;
    ListNode next;
    ListNode(int val) { this.val = val; }
}

// 反转链表
public ListNode reverseList(ListNode head) {
    ListNode prev = null, curr = head;
    while (curr != null) {
        ListNode next = curr.next;
        curr.next = prev;
        prev = curr;
        curr = next;
    }
    return prev;
}

// 判断环：快慢指针
public boolean hasCycle(ListNode head) {
    ListNode slow = head, fast = head;
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
        if (slow == fast) return true;
    }
    return false;
}
```

## 25. 栈与队列

### 有效括号

```java
public boolean isValid(String s) {
    Deque<Character> stack = new ArrayDeque<>();
    for (char c : s.toCharArray()) {
        if (c == '(' || c == '[' || c == '{') {
            stack.push(c);
        } else {
            if (stack.isEmpty()) return false;
            char top = stack.pop();
            if (c == ')' && top != '(') return false;
            if (c == ']' && top != '[') return false;
            if (c == '}' && top != '{') return false;
        }
    }
    return stack.isEmpty();
}
```

### 用队列做二叉树层序遍历

```java
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> res = new ArrayList<>();
    if (root == null) return res;
    Queue<TreeNode> q = new LinkedList<>();
    q.offer(root);
    while (!q.isEmpty()) {
        int size = q.size();
        List<Integer> level = new ArrayList<>();
        for (int i = 0; i < size; i++) {
            TreeNode node = q.poll();
            level.add(node.val);
            if (node.left != null) q.offer(node.left);
            if (node.right != null) q.offer(node.right);
        }
        res.add(level);
    }
    return res;
}
```

## 26. 二叉树递归模板

```java
class TreeNode {
    int val;
    TreeNode left, right;
    TreeNode(int val) { this.val = val; }
}

// 最大深度
public int maxDepth(TreeNode root) {
    if (root == null) return 0;
    return 1 + Math.max(maxDepth(root.left), maxDepth(root.right));
}

// 前序遍历（递归）
public void preorder(TreeNode root, List<Integer> list) {
    if (root == null) return;
    list.add(root.val);
    preorder(root.left, list);
    preorder(root.right, list);
}
```

**递归三要素**：终止条件、当前层逻辑、向下一层传递。

## 27. 滑动窗口模板

```java
// 最长无重复字符子串长度
public int lengthOfLongestSubstring(String s) {
    Map<Character, Integer> last = new HashMap<>();
    int left = 0, max = 0;
    for (int right = 0; right < s.length(); right++) {
        char c = s.charAt(right);
        if (last.containsKey(c) && last.get(c) >= left) {
            left = last.get(c) + 1;
        }
        last.put(c, right);
        max = Math.max(max, right - left + 1);
    }
    return max;
}
```

固定窗口：先扩到窗口大小，再右移 left。

## 28. 二分查找模板

```java
// 在有序数组中找 target 下标，不存在返回 -1
public int binarySearch(int[] nums, int target) {
    int left = 0, right = nums.length - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] == target) return mid;
        if (nums[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}
```

注意：`mid = left + (right - left) / 2` 防溢出。

## 29. 回溯入门：全排列

```java
public List<List<Integer>> permute(int[] nums) {
    List<List<Integer>> res = new ArrayList<>();
    backtrack(nums, new boolean[nums.length], new ArrayList<>(), res);
    return res;
}

private void backtrack(int[] nums, boolean[] used, List<Integer> path, List<List<Integer>> res) {
    if (path.size() == nums.length) {
        res.add(new ArrayList<>(path));
        return;
    }
    for (int i = 0; i < nums.length; i++) {
        if (used[i]) continue;
        used[i] = true;
        path.add(nums[i]);
        backtrack(nums, used, path, res);
        path.remove(path.size() - 1);
        used[i] = false;
    }
}
```

回溯口诀：**做选择 → 递归 → 撤销选择**。

## 34. 高频算法模式汇总

### 34.1 双指针

```java
// 反转数组
public void reverse(int[] arr) {
    int left = 0, right = arr.length - 1;
    while (left < right) {
        int tmp = arr[left];
        arr[left++] = arr[right];
        arr[right--] = tmp;
    }
}
```

### 34.2 滑动窗口

```java
// 无重复字符的最长子串长度
public int lengthOfLongestSubstring(String s) {
    int[] map = new int[128];
    int left = 0, max = 0;
    for (int right = 0; right < s.length(); right++) {
        left = Math.max(left, map[s.charAt(right)]);
        max = Math.max(max, right - left + 1);
        map[s.charAt(right)] = right + 1;
    }
    return max;
}
```

### 34.3 二分查找

```java
public int binarySearch(int[] arr, int target) {
    int left = 0, right = arr.length - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;  // 防溢出
        if (arr[mid] == target) return mid;
        else if (arr[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}
```

```java
// 二叉树前序遍历（根 → 左 → 右）
void preorder(TreeNode root, List<Integer> res) {
    if (root == null) return;            // 终止条件
    res.add(root.val);                    // 处理根
    preorder(root.left, res);             // 递归左子树
    preorder(root.right, res);            // 递归右子树
}
```

## 38. 手把手：快速排序分治（Java 模板）

| 步骤 | 你的动作 | 预期结果 | 若不对 |
|------|----------|----------|--------|
| 1 | 选 `pivot = arr[right]` | 基准值确定 | 空数组直接 return |
| 2 | 双指针 `i` 从左扫，`j` 从右扫 | `i` 找大于 pivot，`j` 找小于 | 边界 `i<j` 时交换 |
| 3 | 交换 pivot 到正确位置 | pivot 左边 ≤ pivot，右边 ≥ | 分区错误则检查比较条件 |
| 4 | 递归 `quickSort(left, pi-1)` 和 `quickSort(pi+1, right)` | 全数组有序 | 忘记递归则只排一段 |
| 5 | 对 `[5,2,8,1,9]` 手跑一遍 | 输出 `1,2,5,8,9` | 对照 §38.1 代码 |

### 38.1 快速排序代码

```java
public void quickSort(int[] arr, int left, int right) {
    if (left >= right) return;
    int pi = partition(arr, left, right);
    quickSort(arr, left, pi - 1);
    quickSort(arr, pi + 1, right);
}

private int partition(int[] arr, int left, int right) {
    int pivot = arr[right];
    int i = left - 1;
    for (int j = left; j < right; j++) {
        if (arr[j] <= pivot) {
            i++;
            swap(arr, i, j);
        }
    }
    swap(arr, i + 1, right);
    return i + 1;
}

private void swap(int[] arr, int a, int b) {
    int t = arr[a]; arr[a] = arr[b]; arr[b] = t;
}
```

**逐行读 `partition`**：

| 行号/代码 | 含义 | 改错会怎样 |
|-----------|------|------------|
| `pivot = arr[right]` | 选最右为基准（可优化随机/三数取中） | 有序数组最坏 O(n²) |
| `i = left - 1` | 小于等于 pivot 区域的右边界 | 初值错则交换乱 |
| `j` 从左到右扫描 | 找应进左区的元素 | 漏扫则分区不对 |
| `arr[j] <= pivot` 则 `i++` 并 swap | 扩大小于区 | 用 `<` 而非 `<=` 稳定性变 |
| 最后 swap pivot 到 `i+1` | pivot 归位 | 不 swap 则 pivot 不在中间 |

**术语（分治 Divide and Conquer）**：拆成小问题、分别解决、合并结果。  
**生活类比**：整理书架——先按某一本书把小的放左大的放右，再左右各自重复。  
**为什么重要**：快排平均 O(n log n)，面试常要求手写思路。  
**本章用到的地方**：§17、§35、§38。

## 39. 动态规划入门三题速记

| 题目 | 状态定义 | 转移 | 复杂度 |
|------|----------|------|--------|
| 爬楼梯 | `dp[i]`=到第 i 阶方案数 | `dp[i]=dp[i-1]+dp[i-2]` | O(n) |
| 打家劫舍 | `dp[i]`=前 i 家最大金额 | 偷 i：`nums[i]+dp[i-2]`；不偷：`dp[i-1]` | O(n) |
| 最大子序和 | `dp[i]`=以 i 结尾最大和 | `dp[i]=max(nums[i], dp[i-1]+nums[i])` | O(n) |

```java
// 爬楼梯 — 空间优化版
public int climbStairs(int n) {
    if (n <= 2) return n;
    int a = 1, b = 2;
    for (int i = 3; i <= n; i++) {
        int c = a + b;
        a = b;
        b = c;
    }
    return b;
}
```

## 41. FAQ

**Q1：要刷多少题？**  
后端初面：40～80 道 Easy/Medium 高频即可；重质量不重数量。

**Q2：和数据结构文件夹区别？**  
数据结构系列讲原理与手写实现；本章给 **Java 面试模板** 和刷题路线。

**Q3：递归总是栈溢出？**  
检查终止条件；深树改迭代 + 栈（层序遍历用队列）。

**Q4：双指针和滑动窗口区别？**  
双指针常指左右夹逼或快慢指针；滑动窗口是维护一段连续区间长度。

**Q5：HashMap 和数组下标？**  
需要 O(1) 查「值→下标」用 HashMap；数组本身按下标 O(1) 访问。

**Q6：面试能查资料吗？**  
现场手写不能；部分公司 OA 可搜索——以目标公司为准。

**Q7：树题为什么高频？**  
同时考递归、队列 BFS、结构理解；是后端初面最爱题型之一。

**Q8：快排最坏情况？**  
已有序且总选最边 pivot → O(n²)；随机 pivot 或三数取中缓解。

**Q9：和 14 章关系？**  
13 练「写对代码」；14 练「设计系统」——面试两轮不同。

**Q10：Java `Arrays.sort` 用什么？**  
基本类型 Dual-Pivot Quicksort；对象 Timsort（归并+插入混合）。

**Q11：空间 O(1) 什么意思？**  
额外空间不随输入规模增长（不含输入本身占的空间）。

**Q12：做不出来怎么办？**  
看题解前自己写 20 分钟；看懂后**隔一天重写**，并用自己的话复述。

### 42.1 自测参考答案

1. 数组：随机访问 O(1)，插入删除慢；链表：插入删除 O(1)，查找 O(n)。
2. 快排平均、归并、堆排、二分相关分治。
3. 终止条件、当前层逻辑、向下一层传递/返回值。
4. 平均 O(1)；最坏 O(n)（全冲突链表）。
5. 相等元素排序后相对顺序不变；归并稳定，快排通常不稳定。
6. BFS 用队列；DFS 用栈或递归系统栈。
7. 遍历数组，查 `target-nums[i]` 是否在 map，不在则 put。
8. 三个：`prev`、`curr`、`next`（或两个指针写法）。
9. 窗口右扩左缩可 O(n)；暴力 O(n³) 或 O(n²) 超时。
10. `if (root == null) return;`
