## 1. 这套资料适合谁

- 想走 **Java 后端** 路线的大学生或转行初学者
- 已学完或并行学习 [前端 HTML/CSS/JS](../../前端学习/HTML%20CSS%20JS/00-学习路线图与说明.md) 的同学（前后端联调会更顺）
- 目标：能写 Spring Boot 接口、操作 MySQL/Redis、理解面试高频考点

**不适合**：已多年微服务架构经验、仅想查某一中间件 API 的资深开发者。

> 本仓库同时提供 [Python 后端](../Python/00-学习路线图与说明.md) 与 [C++ 路线](../C++/00-学习路线图与说明.md)（系统/高性能/算法/**Qt 桌面**，见 [C++ 16](../C++/16-必学技术栈分轨与扩展专题.md)），以及独立 [数据结构](../数据结构/README.md) 模块（原理 + 70 题刷题路线）。**Web 业务选 Java 或 Python，C++ 可作主线或补底层，数据结构配合 13 章刷题。**
>
> 走 **Agent + Java** 路线：完成 **04～07** 后进入 [AI Agent 00](../AIAgent/00-学习路线图与说明.md)；实时通信用 [16 SSE/WebSocket](16-SSE与WebSocket实时通信.md)；测试工程化见 [17 单元测试](17-单元测试与Mockito深入.md)。

## 4. 必备环境与工具

### 4.1 JDK

- 推荐 **JDK 17** 或 **JDK 21**（LTS）
- 验证：

```bash
java -version
# 预期输出（示例）：
# openjdk version "17.0.x" ...
# OpenJDK Runtime Environment ...
# OpenJDK 64-Bit Server VM ...

javac -version
# 预期输出：
# javac 17.0.x
```

若提示 `'java' 不是内部或外部命令`：检查环境变量 `JAVA_HOME` 和 `Path` 是否包含 `%JAVA_HOME%\bin`。

### 4.2 IDE

- 推荐 **IntelliJ IDEA**（Community 版够用）
- 必会：运行 main、断点调试、Maven 导入、HTTP Client 测接口

### 4.3 构建工具

- **Maven** 或 **Gradle**（本资料示例以 Maven 为主）
- 理解 `pom.xml` 里依赖与插件

### 4.4 数据库与中间件（学到对应章节再装）

| 组件 | 用途 | 建议 |
|------|------|------|
| MySQL 8 | 关系型数据库 | 本地或 Docker |
| Redis | 缓存 | Docker 一条命令即可 |
| RabbitMQ | 消息队列 | Docker 或本地安装 |
| Docker | 容器化部署 | 09 篇详讲 |
| Postman / Apifox | 测接口 | 与前端 fetch 对照 |

### 4.5 Git

与前端相同：练习项目从第一天就 `git init`，每天至少一次有意义的 commit。

## 9. 常见问题 FAQ

### Q1：要先学完前端再学后端吗？

不强制，但 **HTTP、JSON、接口概念** 先懂会更顺。可 01~04 前端 + 01~04 后端并行。

### Q2：必须学微服务吗？

初学先把 **单体 Spring Boot + MySQL + Redis** 做透。11、12 篇先建立概念，工作中再深入。

### Q3：算法要刷多少题？

13 篇有建议：数组/哈希/链表/树各一批高频题，**精刷 50~80 道** 比泛刷 500 道更有效。

### Q4：IDEA 报错找不到符号？

检查：JDK 是否配置、Maven 是否 reload、包名是否一致。

### Q5：和前端联调跨域？

后端加 CORS 配置，或开发环境用前端代理。见 04 篇与前端 10 篇。
