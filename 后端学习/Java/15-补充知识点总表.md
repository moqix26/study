## 17. 零基础 FAQ

**Q1：总表要全背吗？**  
不用。标 ⬜ 的优先补；🔶 以上可面试前快速扫。

**Q2：「零基础解释」和正文区别？**  
解释列帮你**认字**；掌握标准列告诉你**要学到哪一步**。

**Q3：01～14 和本表顺序？**  
先按 00 路线图顺序学章节；学完一轮再用 15 查漏。

**Q4：前端同学看后端总表？**  
看 §13 对照 + §11 场景题，联调时够用；深入按 Java 00 路线图。

**Q5：多久过一遍总表？**  
日常不用；面试前 1 周每天 30 分钟 + 考前 30 分钟全文扫。

**Q6：⬜ 太多怎么办？**  
按 §12 优先级先补第一优先级，别试图一天全补。

**Q7：和 14 章重复？**  
14 讲怎么答；15 是索引清单，不重复细节。

**Q8：部署 09 和 Linux 系列？**  
09 是 Java 开发者最小部署；系统运维深读 [Linux 00](../Linux/00-学习路线图与说明.md)。

**Q9：算法要全部 ✅ 吗？**  
13 章 40 题 🔶 即可；总表算法行 ✅=能手写模板。

**Q10：如何记录进度？**
