## 11. 高频口述题（5 分钟版）

### 11.1 用 3 分钟讲清 LoRA

```text
问题：全参微调 7B 要 60G+ 显存。
LoRA：冻结 W，学 ΔW=BA，r≪d，只训 0.1% 参数。
推理：可 merge 回 W' 无额外延迟。
超参：r、alpha、target_modules（qkv）。
```

### 11.2 用 3 分钟讲清 RAG（Python vs Java）

```text
索引：load→split→embed→store
查询：embed query→topK→prompt→LLM
Python：LangChain Retriever（21 章）
Java：Spring AI QuestionAnswerAdvisor（AIAgent 06）
微调不能替代：权重无最新私有文档。
```

### 11.3 用 3 分钟讲清训练到部署

```text
SFT+LoRA（15）→ wandb（22）→ merge → vLLM（20）
或 C++ 引擎（LLMInfra 19）读 safetensors
延迟：Prefill vs Decode（LLMInfra 16）
```

## 19. 闭卷自测

1. LLMPython 与 LLMInfra、AIAgent 各负责什么？
2. LoRA 数学形式与省参原因？
3. DDP 为什么要 `dist.init_process_group`？
4. vLLM 相对 HF generate 的两点优势？
5. wandb 中 config 与 log 区别？
6. CLIP 如何实现 zero-shot？
7. torch.compile 与 TorchScript 谁主推？
8. 24 章 README 必须含什么？
9. 为何 document-level 划分数据集？
10. 口述 LoRA 微调完整流程（10 步内）？

<details>
<summary>参考答案</summary>

1. Python 训练微调实验；C++ CUDA 推理引擎；Java Spring 产品 Agent/RAG API。
2. ΔW=BA，B∈R^{d×r}, A∈R^{r×k}，r≪d；只更新 A,B 参数量小。
3. 多进程间建立通信组以 allreduce 梯度。
4. PagedAttention 减 KV 碎片；continuous batching 提吞吐。
5. config 静态超参；log 动态 step 指标。
6. 各类别 prompt embedding 与图像比相似度 argmax。
7. PyTorch 2 主推 torch.compile（Dynamo+Inductor）。
8. 环境、数据说明、训练/评估/Demo 命令、结果摘要。
9. 防同一文档 QA 泄漏到 val 导致指标虚高。
10. 数据准备→选基座→LoRA 配置→Trainer/wandb→train→val→eval→merge→Demo→（vLLM）部署。

</details>
