## 16. 常见困惑 FAQ

**Q1：Infra 岗要背多少 CUDA？**  
能写 vector add + 解释 Flash/roofline；复杂 kernel 读源码即可。

**Q2：和 Java 15 总表怎么用？**  
Java 15 管 Web 后端；本表管 LLM 底层；全栈 LLM 两份都扫。

**Q3：没 GPU 能面试 Infra 吗？**  
理论+项目设计可；**实操题**需云 GPU 经验，至少 19 章 demo。

**Q4：PagedAttention 要读论文吗？**  
读摘要+图；面试讲 block 分配足够；深挖看 vLLM 源码。

**Q5：TRT-LLM 和 TensorRT 关系？**  
TRT-LLM 基于 TensorRT 构建 LLM engine；别混为一谈。

**Q6：C++ 19/20 还没出文档？**  
路线图已规划路径；本章链接 **planned**，以 11、13 章为主。

**Q7：算法题还要刷吗？**  
Infra 重系统+CUDA；算法 [数据结构 11](../数据结构/09-综合复习/README.md) 保持中等强度。

**Q8：如何证明 🔶→✅？**  
对同事费曼 3 分钟并画数据流；或 mock 面试录音自检。

**Q9：训练岗 vs 推理岗？**  
训练加重 §5 DP/TP/NCCL；推理加重 §4、§8、§9。

**Q10：本表多久过一遍？**  
日常不用；面试前 1 周每天 30 min + 考前全文扫。
