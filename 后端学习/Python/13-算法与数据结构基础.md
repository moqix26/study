## 4. 数组与字符串

### 4.1 特点

- 连续存储，下标 O(1) 访问
- Python `list` 动态数组，末尾 append 均摊 O(1)

### 4.2 双指针模板

```python
def two_sum_sorted(nums: list[int], target: int) -> list[int]:
    left, right = 0, len(nums) - 1
    while left < right:
        s = nums[left] + nums[right]
        if s == target:
            return [left, right]
        if s < target:
            left += 1
        else:
            right -= 1
    return []
```

### 4.3 滑动窗口模板

```python
def length_of_longest_substring(s: str) -> int:
    seen: dict[str, int] = {}
    left = ans = 0
    for right, ch in enumerate(s):
        if ch in seen and seen[ch] >= left:
            left = seen[ch] + 1
        seen[ch] = right
        ans = max(ans, right - left + 1)
    return ans
```

## 5. 哈希表（dict / set）

### 5.1 两数之和

```python
def two_sum(nums: list[int], target: int) -> list[int]:
    idx_map: dict[int, int] = {}
    for i, x in enumerate(nums):
        need = target - x
        if need in idx_map:
            return [idx_map[need], i]
        idx_map[x] = i
    return []
```

### 5.2 频次统计

```python
from collections import Counter

def top_k_frequent(nums: list[int], k: int) -> list[int]:
    return [x for x, _ in Counter(nums).most_common(k)]
```

## 6. 链表

### 6.1 节点定义

```python
class ListNode:
    def __init__(self, val: int = 0, next: "ListNode | None" = None):
        self.val = val
        self.next = next
```

### 6.2 反转链表

```python
def reverse_list(head: ListNode | None) -> ListNode | None:
    prev = None
    while head:
        nxt = head.next
        head.next = prev
        prev = head
        head = nxt
    return prev
```

### 6.3 快慢指针找中点

```python
def middle_node(head: ListNode | None) -> ListNode | None:
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
    return slow
```

### 6.4 判断环

```python
def has_cycle(head: ListNode | None) -> bool:
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow is fast:
            return True
    return False
```

## 7. 栈与队列

### 7.1 有效括号

```python
def is_valid(s: str) -> bool:
    stack: list[str] = []
    pairs = {")": "(", "]": "[", "}": "{"}
    for ch in s:
        if ch in pairs:
            if not stack or stack[-1] != pairs[ch]:
                return False
            stack.pop()
        else:
            stack.append(ch)
    return not stack
```

## 8. 二叉树

### 8.1 节点定义

```python
class TreeNode:
    def __init__(self, val: int = 0, left: "TreeNode | None" = None, right: "TreeNode | None" = None):
        self.val = val
        self.left = left
        self.right = right
```

### 8.2 前序遍历（递归）

```python
def preorder(root: TreeNode | None) -> list[int]:
    if not root:
        return []
    return [root.val] + preorder(root.left) + preorder(root.right)
```

### 8.3 最大深度

```python
def max_depth(root: TreeNode | None) -> int:
    if not root:
        return 0
    return 1 + max(max_depth(root.left), max_depth(root.right))
```

### 8.4 层序遍历

```python
from collections import deque

def level_order(root: TreeNode | None) -> list[list[int]]:
    if not root:
        return []
    q = deque([root])
    ans: list[list[int]] = []
    while q:
        level = []
        for _ in range(len(q)):
            node = q.popleft()
            level.append(node.val)
            if node.left:
                q.append(node.left)
            if node.right:
                q.append(node.right)
        ans.append(level)
    return ans
```

## 9. 二分查找

```python
def binary_search(nums: list[int], target: int) -> int:
    lo, hi = 0, len(nums) - 1
    while lo <= hi:
        mid = (lo + hi) // 2
        if nums[mid] == target:
            return mid
        if nums[mid] < target:
            lo = mid + 1
        else:
            hi = mid - 1
    return -1
```

**变体**：找左边界、找右边界、旋转数组（Medium）。

## 10. 回溯入门

```python
def permute(nums: list[int]) -> list[list[int]]:
    ans: list[list[int]] = []

    def dfs(path: list[int], used: set[int]) -> None:
        if len(path) == len(nums):
            ans.append(path[:])
            return
        for x in nums:
            if x in used:
                continue
            used.add(x)
            path.append(x)
            dfs(path, used)
            path.pop()
            used.remove(x)

    dfs([], set())
    return ans
```

## 13. 面试手撕技巧

1. **先问清**：输入规模、是否有序、能否改原数组、边界空输入
2. **说思路再写**：暴力 → 优化，报复杂度

4. **自测**：`[]`、单元素、重复、负数
5. **不会时**：说暴力方案 + 卡在哪，别沉默

## 14. 常见报错与排查

| 现象 | 原因 | 解决 |
|------|------|------|
| TLE | O(n²) 数据过大 | 换 hash/二分/双指针 |
| MLE | 递归过深 | 改迭代或限制深度 |
| IndexError | 空 list / 越界 | 先判 `if not nums` |
| 链表死循环 | 未判 fast.next | while fast and fast.next |
| KeyError | dict 无 key | 用 .get() 或 in |
| 递归栈溢出 | Python 默认深度 | sys.setrecursionlimit（慎用） |
