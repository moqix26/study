## 4. 数组与字符串

### 4.1 特点

- 连续存储，下标 O(1) 访问
- Python `list` 动态数组，末尾 append 均摊 O(1)

### 4.2 双指针模板

```python
def two_sum_sorted(nums: list[int], target: int) -> list[int]:
    left, right = 0, len(nums) - 1
    while left < right:
        s = nums[left] + nums[right]
        if s == target:
            return [left, right]
        if s < target:
            left += 1
        else:
            right -= 1
    return []
```

### 4.3 滑动窗口模板

```python
def length_of_longest_substring(s: str) -> int:
    seen: dict[str, int] = {}
    left = ans = 0
    for right, ch in enumerate(s):
        if ch in seen and seen[ch] >= left:
            left = seen[ch] + 1
        seen[ch] = right
        ans = max(ans, right - left + 1)
    return ans
```

## 5. 哈希表（dict / set）

### 5.1 两数之和

```python
def two_sum(nums: list[int], target: int) -> list[int]:
    idx_map: dict[int, int] = {}
    for i, x in enumerate(nums):
        need = target - x
        if need in idx_map:
            return [idx_map[need], i]
        idx_map[x] = i
    return []
```

### 5.2 频次统计

from collections import Counter

def top_k_frequent(nums: list[int], k: int) -> list[int]:
    return [x for x, _ in Counter(nums).most_common(k)]

## 6. 链表

### 6.1 节点定义

```python
class ListNode:
    def __init__(self, val: int = 0, next: "ListNode | None" = None):
        self.val = val
        self.next = next
```

### 6.2 反转链表

```python
def reverse_list(head: ListNode | None) -> ListNode | None:
    prev = None
    while head:
        nxt = head.next
        head.next = prev
        prev = head
        head = nxt
    return prev
```

### 6.3 快慢指针找中点

```python
def middle_node(head: ListNode | None) -> ListNode | None:
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
    return slow
```

### 6.4 判断环

```python
def has_cycle(head: ListNode | None) -> bool:
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow is fast:
            return True
    return False
```

def is_valid(s: str) -> bool:
    stack: list[str] = []
    pairs = {")": "(", "]": "[", "}": "{"}
    for ch in s:
        if ch in pairs:
            if not stack or stack[-1] != pairs[ch]:
                return False
            stack.pop()
        else:
            stack.append(ch)
    return not stack

## 8. 二叉树

### 8.1 节点定义

```python
class TreeNode:
    def __init__(self, val: int = 0, left: "TreeNode | None" = None, right: "TreeNode | None" = None):
        self.val = val
        self.left = left
        self.right = right
```

### 8.2 前序遍历（递归）

```python
def preorder(root: TreeNode | None) -> list[int]:
    if not root:
        return []
    return [root.val] + preorder(root.left) + preorder(root.right)
```

### 8.3 最大深度

```python
def max_depth(root: TreeNode | None) -> int:
    if not root:
        return 0
    return 1 + max(max_depth(root.left), max_depth(root.right))
```

### 8.4 层序遍历

```python
from collections import deque

def level_order(root: TreeNode | None) -> list[list[int]]:
    if not root:
        return []
    q = deque([root])
    ans: list[list[int]] = []
    while q:
        level = []
        for _ in range(len(q)):
            node = q.popleft()
            level.append(node.val)
            if node.left:
                q.append(node.left)
            if node.right:
                q.append(node.right)
        ans.append(level)
    return ans
```

## 9. 二分查找

```python
def binary_search(nums: list[int], target: int) -> int:
    lo, hi = 0, len(nums) - 1
    while lo <= hi:
        mid = (lo + hi) // 2
        if nums[mid] == target:
            return mid
        if nums[mid] < target:
            lo = mid + 1
        else:
            hi = mid - 1
    return -1
```

**变体**：找左边界、找右边界、旋转数组（Medium）。

## 10. 回溯入门

```python
def permute(nums: list[int]) -> list[list[int]]:
    ans: list[list[int]] = []

    def dfs(path: list[int], used: set[int]) -> None:
        if len(path) == len(nums):
            ans.append(path[:])
            return
        for x in nums:
            if x in used:
                continue
            used.add(x)
            path.append(x)
            dfs(path, used)
            path.pop()
            used.remove(x)

    dfs([], set())
    return ans
```

## 14. 常见报错与排查

| 现象 | 原因 | 解决 |
|------|------|------|
| TLE | O(n²) 数据过大 | 换 hash/二分/双指针 |
| MLE | 递归过深 | 改迭代或限制深度 |
| IndexError | 空 list / 越界 | 先判 `if not nums` |
| 链表死循环 | 未判 fast.next | while fast and fast.next |
| KeyError | dict 无 key | 用 .get() 或 in |
| 递归栈溢出 | Python 默认深度 | sys.setrecursionlimit（慎用） |

## 17. FAQ

**Q1：Python 刷题和 Java 有什么区别？**  
思路相同；Python 用 `list`/`dict`/`deque`，注意递归深度与 `//` 整除。

**Q2：要刷多少题够面试？**  
本章 **40 道 B 题** 为底线；目标公司更高可加到 80～100。

**Q3：Hard 题要不要做？**  
校招/backend 以 **Easy + Medium** 为主；Hard 有余力再碰。

**Q4：记不住模板怎么办？**  
**理解不变式**（如双指针 left/right 含义），比背代码重要；模板见 §4～§9。

**Q5：TLE 怎么排查？**  
看数据规模：10⁵ 需 O(n) 或 O(n log n)；检查是否嵌套循环 O(n²)。

**Q6：链表题 dummy 节点干什么？**  
简化头结点删除/合并边界，避免 special case。

**Q7：BFS 和 DFS 怎么选？**  
层序/最短步数 → BFS（队列）；路径/回溯/树深度 → DFS（递归/栈）。

**Q8：`heapq` 什么时候用？**  
TopK、合并 K 路有序流、Dijkstra；注意 Python 是小根堆。

**Q9：算法和 backend 项目怎么结合？**  
面试说：「LRU 题帮助理解 Redis 缓存淘汰」「哈希题对应 dict O(1) 查找」。

**Q10：需要背 LeetCode 题号吗？**  
知道经典题号加分（206/146/704）；更重要的是**题型**。

**Q11：递归超限怎么办？**  
改迭代；或 `sys.setrecursionlimit`（慎用）；优先改算法。

**Q12：和 14 章关系？**  
13 练代码逻辑；14 练场景表达——面试两类题都要准备。

10. **综合**：面试官让写 LRU，你会提到哪些数据结构/API？

### 自测参考答案

1. 10⁵ 级别 n 用 O(n)；n log n 通常也可；O(n²) 易 TLE。
2. 平均 O(1)；最坏 O(n)（哈希冲突）。
3. 最近匹配的开括号用 LIFO 栈。
4. 有序（或单调）+ 可索引；确定 mid 后排除一半。
5. BFS→`collections.deque`；DFS→递归或显式栈。
6. 连续子数组/子串满足某条件的最值或计数。
7. 遍历 nums，`need = target - x`，查 dict 是否见过 need。
8. 通常 `prev`、`curr` 两个（或三个含 dummy）。
9. 队列存节点，逐层 for 循环 len(queue) 处理一层。
10. OrderedDict 或 dict+双向链表；get/put O(1)；联系 07 章缓存。
