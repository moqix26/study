## 1. 为什么重载运算符

```cpp
Vector c = a + b;  // 比 a.add(b) 更自然
```

| 限制 | 说明 |
| --- | --- |
| 不能重载 | `. :: ?: sizeof` 等 |
| 不能改 | 优先级、结合性、操作数个数 |
| 至少一侧 | 必须是用户类型 |

## 2. 成员 vs 非成员

| 运算符 | 通常形式 |
| --- | --- |
| `= [] () ->` | 成员 |
| `+ - == <<` | 成员或非成员均可 |
| `<<` 左操作数是 ostream | 非成员 |

## 3. 友元入门

### 术语三件套：友元

| 维度 | 内容 |
| --- | --- |
| **定义** | 非成员函数或类，被授权访问 private/protected。 |
| **生活类比** | 持 VIP 卡进后台。 |
| **为什么重要** | 实现 operator<<、跨类协作。 |
| **本章用到** | §3 |

```cpp
class Vector {
    friend std::ostream& operator<<(std::ostream& os, const Vector& v);
public:
    Vector(double x, double y) : x_(x), y_(y) {}
private:
    double x_, y_;
};
```

## 4. 算术运算符

```cpp
Vector operator+(Vector a, const Vector& b) { a += b; return a; }
```

## 5. 比较运算符

```cpp
bool operator==(const Vector& a, const Vector& b) { return a.x_==b.x_ && a.y_==b.y_; }
```

## 6. 输入输出流

```cpp
std::istream& operator>>(std::istream& is, Vector& v) { return is >> v.x_ >> v.y_; }
```

## 7. 类型转换运算符

```cpp
explicit operator double() const { return num_/den_; }
```

**explicit** 防止隐式转换意外。
## 8. Vector 类完整示例

```cpp
#include <iostream>
#include <cmath>
class Vector {
public:
    Vector(double x = 0, double y = 0) : x_(x), y_(y) {}
    Vector& operator+=(const Vector& rhs) {
        x_ += rhs.x_; y_ += rhs.y_; return *this;
    }
    double length() const { return std::hypot(x_, y_); }
    friend Vector operator+(Vector lhs, const Vector& rhs) {
        lhs += rhs; return lhs;
    }
    friend bool operator==(const Vector& a, const Vector& b) {
        return a.x_ == b.x_ && a.y_ == b.y_;
    }
    friend std::ostream& operator<<(std::ostream& os, const Vector& v) {
        return os << "Vector(" << v.x_ << ", " << v.y_ << ")";
    }
private:
    double x_, y_;
};
int main() {
    Vector v1{3, 4}, v2{1, 2};
    std::cout << v1 + v2 << " len=" << v1.length() << '\n';
    return 0;
}
```

## FAQ

**Q：哪些运算符不能重载？**  
A：`. :: ?: sizeof` 等；`=` 只能成员重载。

**Q：为何 << 常做友元？**  
A：左操作数是 ostream，不能是 Vector 成员。

**Q：+= 应返回什么？**  
A：通常 `*this` 引用。

**Q：explicit 转换作用？**  
A：禁止隐式转换。

**Q：友元破坏封装吗？**  
A：友元是授权接口；应少而精。

**Q：成员 operator+ 缺点？**  
A：难支持 `2 * v`。

**Q：重载能改优先级吗？**  
A：不能。

**Q：operator bool 为何慎用？**  
A：可能意外参与整数提升；用 explicit。

**Q：C++20 <=> 作用？**  
A：一次生成比较族（了解即可）。

**Q：Vector + 为何传值 lhs？**  
A：拷贝再 +=，不修改原对象。
