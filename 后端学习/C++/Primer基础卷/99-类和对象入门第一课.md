## 本章与上一章的关系

[98 章](98-结构体联合与枚举入门.md) 用 struct 打包数据；本章升级为 class，把操作与数据绑在一起，并用 private 保护不变式。

## 1. 为什么需要类

用 struct 只能放数据；操作散落在外部函数，容易不一致：
```cpp
struct StockBad { std::string symbol; int shares; double price; };
```

> **生活类比**：ATM — 你不能直接改内部现金计数，只能按接口「取款」「查询」。

### 术语三件套：封装

| 维度 | 内容 |
| --- | --- |
| **定义** | 把数据与操作数据的代码组织在类中，并隐藏实现细节。 |
| **生活类比** | 自动柜员机：外面只有按键，内部钞箱不直接暴露。 |
| **为什么重要** | 减少耦合、保证不变式。 |
| **本章用到** | §1～§3 |

## 2. 成员变量与成员函数

```cpp
class Clock {
public:

private:
    int hour_{0}, minute_{0};
};
```

## 3. 访问控制 public / private / protected

| specifier | 类内 | 类外 |
| --- | --- | --- |
| public | ✓ | ✓ |
| private | ✓ | ✗ |
| protected | ✓ | 派生类 |

## 4. 构造函数

### 术语三件套：构造函数

| 维度 | 内容 |
| --- | --- |
| **定义** | 与类同名、无返回类型，创建对象时自动调用。 |
| **生活类比** | 产品出厂检测线。 |
| **为什么重要** | 保证对象创建即有效。 |
| **本章用到** | §4 |

```cpp
class Stock {
public:
    Stock(const std::string& sym, int sh, double pr)
        : symbol_(sym), shares_(sh), price_(pr) {}
private:
    std::string symbol_;
    int shares_;
    double price_;
};
```

**成员初始化列表**比在函数体赋值更高效（const/引用成员必须用它）。
## 5. 析构函数

```cpp
class FileHolder { public: FileHolder(); ~FileHolder(); };
```

对象销毁时自动调用 `~ClassName()`，用于释放资源（RAII 预告）。
## 6. this 指针

```cpp
class Counter {
public:
    Counter& add(int n) { value_ += n; return *this; }
private:
    int value_{0};
};
```

每个非 static 成员函数隐式有 `this`（类型 `T* const`）。
## 7. 声明与实现分离

```cpp
// stock.h
#pragma once
#include <string>
class Stock {
public:
    Stock(const std::string& sym, int sh, double pr);
    double total_val() const;
private:
    std::string symbol_;
    int shares_;
    double price_;
};
// stock.cpp
#include "stock.h"
Stock::Stock(const std::string& sym, int sh, double pr)
    : symbol_(sym), shares_(sh), price_(pr) {}
double Stock::total_val() const { return shares_ * price_; }
```

## 8. Stock 完整示例

```cpp
#include <iostream>
#include <string>
class Stock {
public:
    Stock() = default;
    Stock(const std::string& sym, int sh, double pr)
        : symbol_(sym), shares_(sh), price_(pr) {}
    void buy(int n, double p) {
        if (n <= 0) return;
        shares_ += n;
        price_ = p;
    }
    double total() const { return shares_ * price_; }
    void show() const {
        std::cout << symbol_ << ' ' << shares_ << '@' << price_
                  << " = " << total() << '\n';
    }
private:
    std::string symbol_;
    int shares_{0};
    double price_{0.0};
};
int main() {
    Stock s("CPP", 10, 100.0);
    s.buy(5, 105.0);
    s.show();
    return 0;
}
```

## FAQ

**Q：class 与 struct 区别？**  
A：默认访问：class private，struct public；class 更强调封装。

**Q：构造函数能有返回值吗？**  
A：不能；与类同名且无返回类型。

**Q：析构函数何时调用？**  
A：对象生命周期结束：离开作用域、delete 等。

**Q：this 能改指向吗？**  
A：不能；this 是 T* const。

**Q：= default 含义？**  
A：请求编译器生成默认特殊成员。

**Q：头文件为何要 #pragma once？**  
A：防止重复 include 导致重复定义。

**Q：const 成员函数意义？**  
A：承诺不修改对象；const 对象只能调 const 成员。

**Q：初始化列表何时必须？**  
A：const 成员、引用成员、无默认构造的成员。

**Q：Stock price 为何 private？**  
A：防止外部写成负数破坏不变式。

**Q：类外定义为何要 Class::？**  
A：标明函数所属类作用域。
