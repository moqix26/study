## 1. iostream 体系

```cpp
#include <iostream>   // cin, cout, cerr, clog
#include <fstream>    // ifstream, ofstream, fstream
#include <sstream>    // istringstream, ostringstream
```

| 类 | 用途 |
|----|------|
| `istream` | 输入 |
| `ostream` | 输出 |
| `iostream` | 双向 |
| `ifstream` | 文件读 |
| `ofstream` | 文件写 |
| `fstream` | 文件读写 |
| `istringstream` | 从 string 读 |
| `ostringstream` | 写入 string |


### 术语三件套：流（stream）

| 维度 | 内容 |
|------|------|
| **定义** | 字节或字符的连续序列，统一了终端、文件、内存缓冲的 IO 接口。 |
| **生活类比** | 自来水管：水（数据）从一端流到另一端。 |
| **为什么重要** | C++ IO 核心抽象；与 C stdio 不同。 |
| **本章用到** | §1 |

## 3. 操纵符（manipulator）

```cpp
#include <iostream>
#include <iomanip>

int main() {
    double pi = 3.1415926;
    std::cout << std::fixed << std::setprecision(2) << pi << '\n';  // 3.14
    std::cout << std::setw(10) << 42 << '\n';  // 右对齐宽度 10
    std::cout << std::hex << 255 << '\n';      // ff
    std::cout << std::dec;  // 恢复十进制
    return 0;
}
```

| 操纵符 | 作用 |
|--------|------|
| `std::endl` | 换行 + 刷新缓冲区 |
| `\n` | 换行（ often 更快） |
| `setprecision(n)` | 浮点精度 |
| `fixed` / `scientific` | 固定/科学计数 |
| `setw(n)` | 字段宽度 |
| `setfill(c)` | 填充字符 |
| `left` / `right` | 对齐 |

## 4. 文件流 fstream

### 4.1 写文件

```cpp
#include <fstream>
#include <string>

int main() {
    std::ofstream fout("out.txt");
    if (!fout) {
        std::cerr << "无法打开文件\n";
        return 1;
    }
    fout << "第一行\n第二行\n";
    fout.close();
    return 0;
}
```

### 4.2 读文件

```cpp
#include <fstream>
#include <string>
#include <iostream>

int main() {
    std::ifstream fin("out.txt");
    std::string line;
    while (std::getline(fin, line))
        std::cout << line << '\n';
    return 0;
}
```

### 4.3 文件 open 模式

```cpp
std::fstream fs;
fs.open("data.bin", std::ios::in | std::ios::out | std::ios::binary);
```

| 模式 | 含义 |
|------|------|
| `ios::in` | 读 |
| `ios::out` | 写（截断） |
| `ios::app` | 追加 |
| `ios::ate` | 打开后定位到尾 |
| `ios::binary` | 二进制 |
| `ios::trunc` | 截断 |

## 5. 二进制 IO

```cpp
#include <fstream>

struct Point { int x; int y; };

int main() {
    Point p{10, 20};
    std::ofstream ofs("pts.bin", std::ios::binary);
    ofs.write(reinterpret_cast<const char*>(&p), sizeof(p));
    ofs.close();

    Point q{};
    std::ifstream ifs("pts.bin", std::ios::binary);
    ifs.read(reinterpret_cast<char*>(&q), sizeof(q));
    return 0;
}
```


#### 常见错误：文本模式读二进制

**错误写法：**

```cpp
std::ifstream f("img.bin");  // 未 binary
```

**正确写法：**

```cpp
std::ifstream f("img.bin", std::ios::binary);
```

**原因**：Windows 文本模式可能转换换行，破坏二进制数据。

## 9. 输入缓冲与 cin 陷阱

```cpp
int age;
char grade;
std::cin >> age;
std::cin >> grade;  // 可能读到换行符，而非期望的 'A'

// 正确：跳过空白
std::cin >> age;
std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');
std::cin >> grade;
```


#### 常见错误：getline 前有 cin >>

**错误写法：**

```cpp
int n; std::cin >> n;
std::string line;
std::getline(std::cin, line);  // line 可能为空
```

**正确写法：**

```cpp
std::cin >> n;
std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');
std::getline(std::cin, line);
```

**原因**：>> 后缓冲区残留换行，getline 立即读到空行。


## 10. 输出格式综合示例

```cpp
#include <iostream>
#include <iomanip>
#include <vector>

int main() {
    std::vector<double> scores{88.5, 92.0, 76.5};
    std::cout << std::fixed << std::setprecision(1);
    std::cout << std::left << std::setw(8) << "序号" << std::setw(10) << "分数" << '\n';
    std::cout << std::string(18, '-') << '\n';
    for (std::size_t i = 0; i < scores.size(); ++i)
        std::cout << std::setw(8) << (i + 1) << std::setw(10) << scores[i] << '\n';
    return 0;
}
```

## 11. 文件读写模式详解

```cpp
// 追加日志
std::ofstream log("app.log", std::ios::app);
log << "新的一条记录\n";

// 读写同一文件
std::fstream io("data.txt", std::ios::in | std::ios::out);
io << "overwrite line 1\n";
io.seekg(0);
std::string line;
std::getline(io, line);
```

| 组合 | 行为 |
|------|------|
| `out` 默认 | 创建或截断 |
| `app` | 末尾追加 |
| `in \| out \| trunc` | 读写并清空 |
| `binary` | 字节级，不转换换行 |

## 12. 二进制文件与结构体数组

```cpp
#include <fstream>
#include <vector>

struct Record { int id; double value; };

void writeRecords(const std::string& path, const std::vector<Record>& recs) {
    std::ofstream ofs(path, std::ios::binary);
    for (const auto& r : recs)
        ofs.write(reinterpret_cast<const char*>(&r), sizeof(r));
}

std::vector<Record> readRecords(const std::string& path) {
    std::vector<Record> out;
    std::ifstream ifs(path, std::ios::binary);
    Record r;
    while (ifs.read(reinterpret_cast<char*>(&r), sizeof(r)))
        out.push_back(r);
    return out;
}
```

## 13. stringstream 高级用法

```cpp
#include <sstream>
#include <string>

std::vector<std::string> split(const std::string& line, char delim) {
    std::vector<std::string> tokens;
    std::istringstream iss(line);
    std::string tok;
    while (std::getline(iss, tok, delim))
        if (!tok.empty()) tokens.push_back(tok);
    return tokens;
}

// CSV 一行
auto fields = split("Alice,90,Beijing", ',');
```

## 14. 与 104 章 vector 读文件完整示例

```cpp
#include <fstream>
#include <vector>
#include <string>
#include <iostream>

int main() {
    std::ifstream fin("names.txt");
    if (!fin) {
        std::cerr << "无法打开 names.txt\n";
        return 1;
    }
    std::vector<std::string> names;
    std::string line;
    while (std::getline(fin, line))
        names.push_back(line);

    for (const auto& n : names)
        std::cout << n << '\n';
    return 0;
}
```

## FAQ

**Q：endl vs \n？**  
A：endl 刷新缓冲；大量输出时 \n 更快。

**Q：路径反斜杠？**  
A：Windows 用 `"path\\file.txt"` 或 R"(path\file.txt)"。

**Q：追加写？**  
A：`ios::app` 或 `ofstream f(name, ios::app)`。

**Q：二进制可移植？**  
A：struct 直接 write 可能因对齐/_endian 不 portable；用序列化库。

**Q：stringstream 清空？**  
A：`str("")` + `clear()`。

**Q：getline 与 >>？**  
A：>> 跳过空白；getline 读整行含空格。

**Q：宽字符？**  
A：wcout/wcin，见 locale 扩展。

**Q：与 C fopen？**  
A：优先 fstream RAII 自动 close。
