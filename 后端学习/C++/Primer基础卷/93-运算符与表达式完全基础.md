## §14 副作用与求值顺序

> 生活类比：改变量或 IO 是副作用；一行别对同一变量改多次。

### 术语三件套

| 维度 | 内容 |
| --- | --- |
| **定义** | `i++ + i++` 未定义行为。 |
| **生活类比** | 改变量或 IO 是副作用；一行别对同一变量改多次。 |
| **为什么重要** | 清晰代码胜于钻标准空子。 |
| **本章用到** | §14 |

### 最小可运行例子

```cpp
int i=0; int j=i++;//j=0,i=1
```

**逐行读代码**：
1. 仅一次修改规则

#### 循序渐进练习

用独立语句代替复杂逗号表达式。

#### 本节小结

- 拆行是最安全的优先级工具。
