## 本卷定位

面向 **完全没碰过 C/C++** 的学习者，像读《C++ Primer Plus》一样从"为什么需要"讲起：每章配生活类比、最小可编译例子、常见错误、循序渐进小节、章末编程练习 5–8 题+答案、FAQ、闭卷自测 10 题+答案。

学完本卷（约 4–6 周）后，回到上级目录的 **01–08 紧凑基础教程**巩固，再进 **37–50 语言深度专题**深挖，最后走 **09+ 工程轨**。
