## 数组与字符串进阶：栈数组生命周期

函数内 `int a[10]` 函数结束销毁；勿返回局部数组指针。

```cpp
void f(){int a[10];}
```

**警告**：返回 &a[0] 悬垂

---

## 数组与字符串进阶：部分初始化零填充

int a[5]={1,2}; 后三个为 0。

```cpp
int a[5]={1,2};
```

**警告**：以为未定义

---

## 数组与字符串进阶：C 串字面量只读

修改 "hi"[0]='H' 未定义。

```cpp
char*p="hi"; //勿写 *p='H'
```

**警告**：应复制到数组

---

## 数组与字符串进阶：strncpy 与截断

strncpy 未必加 \0；需手动补。

```cpp
char d[4]; strncpy(d,"hello",3); d[3]='\0';
```

**警告**：以为自动结尾

---

## 数组与字符串进阶：strcmp 返回值三态

<0 小于，0 相等，>0 大于。

```cpp
strcmp("ab","ac")<0
```

**警告**：与返回 bool 混淆

---

## 数组与字符串进阶：string 与 C 串互转

string s="hi"; s.c_str() 得 const char*。

std::string s(buf);
```

**警告**：c_str 指针勿长期保存

## 数组与字符串进阶：getline 与 cin 混用

>> 后残留换行；下次 getline 读空行。

cin>>n; cin.ignore(); getline...
```
