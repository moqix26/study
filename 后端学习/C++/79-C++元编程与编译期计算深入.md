## 1. TMP 基础

```cpp
template<int N>
struct Fact { static constexpr int value = N * Fact<N-1>::value; };
template<> struct Fact<0> { static constexpr int value = 1; };
```

## 2. SFINAE

```cpp
#include <type_traits>
template<typename T, typename = std::enable_if_t<std::is_integral_v<T>>>
void foo(T) { /* 整数 */ }

// C++20 Concepts 替代
template<std::integral T>
void foo2(T) {}
```

## 3. constexpr 函数

```cpp
constexpr int fib(int n) {
    return (n <= 1) ? n : fib(n-1) + fib(n-2);
}
static_assert(fib(10) == 55);
```

## 6. void_t 检测成员
```cpp
template<typename, typename = void> struct has_foo : std::false_type {};
template<typename T>
struct has_foo<T, std::void_t<decltype(std::declval<T>().foo())>> : std::true_type {};
```

## 7. if constexpr 编译期分派
```cpp
template<typename T>
auto getValue(T& t) {
    if constexpr (std::is_pointer_v<T>) return *t;
    else return t;
}
```

## 8. 编译期斐波那契（模板 vs constexpr 循环）
```cpp
constexpr int fib_c(int n) {
    if (n <= 1) return n;
    int a = 0, b = 1;
    for (int i = 2; i <= n; ++i) { int c = a + b; a = b; b = c; }
    return b;
}
static_assert(fib_c(20) == 6765);
```

## 9. 自定义 is_same
```cpp
template<typename, typename> struct is_same_t : std::false_type {};
template<typename T> struct is_same_t<T,T> : std::true_type {};
```

template<typename... Args>

## 常见问题 FAQ

1. **TMP 与运行时？** 编译期完成，二进制含专用实例。
2. **SFINAE 含义？** 替换失败不是错误。
3. **Concepts 优势？** 报错可读，约束清晰。
4. **与 06 章？** 06 入门；79 深入 traits/编译期。
5. **constexpr 极限？** C++20 可分配/容器。
6. **fold 左/右？** (args+...) vs (...+args) 结合性。
7. **fib 编译慢？** 用 constexpr 循环或模板递推。
8. **type traits 来源？** 标准 <type_traits> 优先。
9. **何时不用 TMP？** 运行时多态足够且编译时间爆炸。
10. **if constexpr？** 丢弃不可达分支，替代 SFINAE。
