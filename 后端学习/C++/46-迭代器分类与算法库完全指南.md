## 1. 五种迭代器（C++17 前四类 + 五类）

| 分类 | 能力 | 典型 |
|---|---|---|
| 输入 Input | 读一次，++ | istream_iterator |
| 输出 Output | 写一次，++ | ostream_iterator |
| 前向 Forward | 多读，++ | forward_list |
| 双向 Bidirectional | +-+ | list, map, set |
| 随机访问 Random | +-, [], 算术 | vector, deque |

## 2. insert 与 erase

```cpp
std::vector<int> v = {1,2,3,4,5};
auto it = v.insert(v.begin() + 2, 99);  // 返回指向首插入元素
v.erase(v.begin() + 1, v.begin() + 3);  // 半开区间 [first, last)
```

**erase 返回值**：指向 **下一元素** 的有效迭代器——[44 章](44-vector-deque-string容器原理与实务.md) 循环删除必用。

---

## 3. sort 与比较

```cpp
#include <algorithm>
std::sort(v.begin(), v.end());
std::sort(v.begin(), v.end(), [](int a, int b) { return a > b; });
```

要求 **随机访问迭代器**；比较 **严格弱序**。[45 章](45-关联容器与哈希容器完全指南.md) 自定义 Compare 同要求。

---

## 4. 二分查找

```cpp
std::sort(v.begin(), v.end());
auto it = std::lower_bound(v.begin(), v.end(), 42);  // 首个 >= 42
auto it2 = std::upper_bound(v.begin(), v.end(), 42); // 首个 > 42
bool found = std::binary_search(v.begin(), v.end(), 42);
```

**前置**：有序；map 自带 `lower_bound`。

## 5. 谓词与 remove-erase

```cpp
v.erase(std::remove_if(v.begin(), v.end(),
    [](int x) { return x % 2 == 0; }), v.end());
```

`remove` **不缩短容器**；与 erase 组合才是删除。

## 8.1 算法练习 1

```cpp
#include <algorithm>
#include <vector>
#include <iostream>

void algo_drill_1() {
    std::vector<int> v(1, 1+5, 1+10, 1+3, 1+7);
    std::sort(v.begin(), v.end());
    auto it = std::lower_bound(v.begin(), v.end(), 1+5);
    if (it != v.end()) std::cout << "lb=" << *it << '\n';
    v.erase(std::remove_if(v.begin(), v.end(),
        [](int x) { return x < 1; }), v.end());
}
```

**迭代器分类 1**：本练习用 **随机访问**；list 上 sort 需 `list::sort` 成员。

**失效 1**：erase/remove 后勿用旧迭代器；[44 章](44-vector-deque-string容器原理与实务.md)。

## 附录 A：高频面试问答

### A.1 sort 复杂度？

O(n log n) 比较；Introsort。

### A.2 list 能 std::sort 吗？

不能；无随机访问；用成员 sort。

### A.3 remove 为何不改 size？

只 **覆盖** 元素，返回新 logical end。

### A.4 istream_iterator？

输入迭代器读流；懒求值陷阱。

### A.5 算法是否检查迭代器合法？

一般不；UB 若越界。
