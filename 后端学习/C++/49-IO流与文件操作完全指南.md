## 2. iostream 架构

```text
                    ios_base
                       │
              ┌────────┴────────┐
           istream           ostream
              │                   │
         ifstream            ofstream
              └────────┬────────┘
                    fstream
```

| 类 | 用途 |
|----|------|
| `std::cin` | 标准输入 |
| `std::cout` | 标准输出 |
| `std::cerr` | 标准错误（无缓冲倾向） |
| `std::ifstream` | 读文件 |
| `std::ofstream` | 写文件 |
| `std::fstream` | 读写文件 |
| `std::stringstream` | 内存字符串流 |

## 3. cin 与 cout 基础

```cpp
#include <iostream>
#include <string>

int main() {
    int age;
    std::string name;
    std::cout << "Enter name and age: ";
    std::cin >> name >> age;
    std::cout << name << " is " << age << " years old\n";
    return 0;
}
```

**陷阱**：`cin >>` 后残留 `\n`，接 `getline` 会读空行：

```cpp
std::cin >> age;
std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');
std::getline(std::cin, line);
```

## 4. 操纵符与 iomanip

```cpp
#include <iomanip>
#include <iostream>

int main() {
    double pi = 3.1415926535;
    std::cout << std::fixed << std::setprecision(2) << pi << '\n';
    std::cout << std::setw(10) << std::setfill('0') << 42 << '\n';
    std::cout << std::hex << std::showbase << 255 << '\n';
    return 0;
}
```

| 操纵符 | 作用 |
|--------|------|
| `std::endl` | 换行 + **flush**（性能敏感用 `'\n'`） |
| `std::fixed` | 固定小数 |
| `std::setprecision(n)` | 精度 |
| `std::setw(w)` | 字段宽度 |

## 5. 文件流 fstream

```cpp
#include <fstream>
#include <iostream>
#include <string>

int main() {
    std::ofstream out("out.txt");
    if (!out) {
        std::cerr << "open failed\n";
        return 1;
    }
    out << "hello\n";
    out.close();  // 也可依赖析构

    std::ifstream in("out.txt");
    std::string line;
    while (std::getline(in, line)) {
        std::cout << line << '\n';
    }
    return 0;
}
```

## 9. 二进制 IO

```cpp
#include <fstream>

struct Header {
    uint32_t magic;
    uint32_t version;
};

void write_bin(const char* path) {
    std::ofstream out(path, std::ios::binary);
    Header h{0xDEADBEEF, 1};
    out.write(reinterpret_cast<const char*>(&h), sizeof(h));
}

void read_bin(const char* path) {
    std::ifstream in(path, std::ios::binary);
    Header h{};
    in.read(reinterpret_cast<char*>(&h), sizeof(h));
}
```

注意：**结构体 padding**、** endian**、跨平台协议用明确字节序（见 18 章对齐）。

## 10. 流状态与异常

```cpp
std::ifstream in("missing.txt");
if (in.fail()) { /* 打开或读失败 */ }

in.exceptions(std::ifstream::failbit | std::ifstream::badbit);
try {
    int x;
    in >> x;
} catch (const std::ios_base::failure& e) {
    std::cerr << e.what() << '\n';
}
```

| 状态 | 含义 |
|------|------|
| goodbit | 正常 |
| eofbit | 到达文件尾 |
| failbit | 逻辑错误 |
| badbit | 严重错误 |

## 13. 常见错误

| # | 错误 | 修复 |
|---|------|------|
| 1 | 不检查 `open` | `if (!fs)` |
| 2 | `cin>>` 后直接 `getline` | `ignore` 换行 |
| 3 | 文本模式读二进制 | 加 `binary` |
| 4 | 结构体直接 fwrite 跨平台 | 序列化协议 |
| 5 | 多线程共用一个流 | 加锁或每线程流 |
