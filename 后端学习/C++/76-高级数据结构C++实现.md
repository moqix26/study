## 1. 跳表 Skip List

### 1.1 原理

```mermaid
graph TD
  L3[头--远跳]
  L2[中层]
  L1[底层链表]
```

期望层数：几何分布 p=1/2；查找 O(log n)，插入均摊 O(log n)。
Redis ZSET 底层 ziplist 或 **skiplist**（元素多且长时）。

### 1.2 完整 C++ 实现

```cpp
#include <random>
#include <vector>
#include <optional>

template<typename K, typename V, int MaxLevel = 16, double P = 0.5>
class SkipList {
    struct Node {
        K key; V value;
        std::vector<Node*> forward;
        Node(int lvl, K k, V v) : key(k), value(v), forward(lvl, nullptr) {}
    };
    Node* head_;
    int level_;
    std::mt19937 rng_{std::random_device{}()};
    int randomLevel() {
        int lvl = 1;
        while ((double)rng_()/rng_.max() < P && lvl < MaxLevel) ++lvl;
        return lvl;
    }
public:
    SkipList() : head_(new Node(MaxLevel, K{}, V{})), level_(1) {}
    ~SkipList() { /* 遍历删除 */ }
    std::optional<V> get(const K& key) {
        Node* cur = head_;
        for (int i = level_-1; i >= 0; --i)
            while (cur->forward[i] && cur->forward[i]->key < key)
                cur = cur->forward[i];
        cur = cur->forward[0];
        if (cur && cur->key == key) return cur->value;
        return std::nullopt;
    }
    void insert(const K& key, const V& val); // 见 §7 完整实现
    bool erase(const K& key) { /* 标准跳表删除 */ return false; }
};
```

## 7. 跳表 insert/erase 完整实现

```cpp
void insert(const K& key, const V& val) {
    std::vector<Node*> update(MaxLevel, head_);
    Node* cur = head_;
    for (int i = level_ - 1; i >= 0; --i) {
        while (cur->forward[i] && cur->forward[i]->key < key)
            cur = cur->forward[i];
        update[i] = cur;
    }
    cur = cur->forward[0];
    if (cur && cur->key == key) { cur->value = val; return; }
    int lvl = randomLevel();
    if (lvl > level_) {
        for (int i = level_; i < lvl; ++i) update[i] = head_;
        level_ = lvl;
    }
    Node* node = new Node(lvl, key, val);
    for (int i = 0; i < lvl; ++i) {
        node->forward[i] = update[i]->forward[i];
        update[i]->forward[i] = node;
    }
}

bool erase(const K& key) {
    std::vector<Node*> update(MaxLevel, nullptr);
    Node* cur = head_;
    for (int i = level_ - 1; i >= 0; --i) {
        while (cur->forward[i] && cur->forward[i]->key < key)
            cur = cur->forward[i];
        update[i] = cur;
    }
    cur = cur->forward[0];
    if (!cur || cur->key != key) return false;
    for (int i = 0; i < level_; ++i) {
        if (update[i]->forward[i] != cur) break;
        update[i]->forward[i] = cur->forward[i];
    }
    delete cur;
    while (level_ > 1 && !head_->forward[level_ - 1]) --level_;
    return true;
}
```

## 10. 布隆过滤器

```cpp
#include <bitset>
#include <functional>
class BloomFilter {
    std::vector<uint64_t> bits_;
    size_t m_; int k_;
    size_t h(const std::string& s, int seed) const {
        return std::hash<std::string>{}(s + std::to_string(seed)) % m_;
    }
public:
    BloomFilter(size_t m, int k) : bits_((m+63)/64), m_(m), k_(k) {}
    void add(const std::string& s) {
        for (int i = 0; i < k_; ++i) {
            size_t pos = h(s, i);
            bits_[pos/64] |= (1ULL << (pos%64));
        }
    }
    bool maybe(const std::string& s) const {
        for (int i = 0; i < k_; ++i) {
            size_t pos = h(s, i);
            if (!(bits_[pos/64] & (1ULL << (pos%64)))) return false;
        }
        return true;
    }
};
```

误判率 ≈ `(1 - e^(-kn/m))^k`；**不漏判**（no false negative）。

## 11. Trie 完整实现

```cpp
struct TrieNode {
    std::array<TrieNode*, 26> next{};
    bool end = false;
    ~TrieNode() { for (auto* p : next) delete p; }
};
class Trie {
    TrieNode* root = new TrieNode;
public:
    void insert(const std::string& w) {
        auto* p = root;
        for (char ch : w) {
            int i = ch - 'a';
            if (!p->next[i]) p->next[i] = new TrieNode;
            p = p->next[i];
        }
        p->end = true;
    }
    bool search(const std::string& w) const {
        auto* p = root;
        for (char ch : w) {
            int i = ch - 'a';
            if (!p->next[i]) return false;
            p = p->next[i];
        }
        return p->end;
    }
    bool startsWith(const std::string& pre) const { /* 同 search 不要求 end */ return true; }
};
```

### 15. 红黑树 vs AVL

### 16. B+ 树页大小

### 17. Counting Bloom

每 counter 减删元素；空间大于经典 Bloom。

### 18. Radix Tree

## 常见问题 FAQ

1. **跳表 vs 红黑树？** 跳表实现简单期望平衡；红黑树最坏 O(log n) 严格。
2. **B+ 为何叶子链？** 范围扫描顺序 IO。
3. **布隆能删吗？** 经典不能；Counting Bloom 可以。
4. **与 28 章？** 28 vector/string；76 索引结构。
5. **Redis ZSET？** skiplist + dict 双索引。
6. **Trie 内存？** 稀疏用 map 子节点；密集用数组。
7. **误判率公式？** m 位 n 元素 k hash。
8. **手写面试？** 跳表/Trie 常考；红黑树说原理即可。
9. **与 45 章？** 45 哈希 O(1)；76 有序/log 结构。
10. **持久化？** B+ 盘页对齐；跳表多用于内存。
