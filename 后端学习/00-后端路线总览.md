## 5. Go 路线 vs Java 路线：复用与跳过

| 模块 | Go 路线怎么做 |
|------|----------------|
| MySQL | 读 **Java/06** 学原理 → **Go/07** 写 GORM |
| Redis | 读 **Java/07** 学原理 → **Go/08** 写 go-redis |
| 部署 | **Go/13** 为主 + **Linux 12～14** 补系统 |
| Spring/MyBatis | **跳过** |
| 算法 | **数据结构 13** + 热题 100（Go 写） |
| 项目 | **Go/10～11** + **系统设计/08** |
