### 1.2 TypeScript 为何独立成文件夹

- **不放进 HTML CSS JS**：TS 是编译型语言 + 类型系统，体量与 JS 基础并列，塞进去会打乱 00～14 编号。
- **不拆成 Vue-TS / React-TS 两套**：基础类型、泛型、tsconfig 只讲一遍；框架差异放在 07～08 章分轨。
- **不放后端学习**：TS 是前端/Node 生态语言，与 Java 无直接对应关系。

**2026 暑假主线**：**Go 01～15** + 数据结构 13 + Linux + 计网 + Java/06、07（原理）。见 [Go 00](后端学习/Go/00-学习路线图与说明.md)。

**2026 默认长期储备**：**C++ 01～36**（+ 数据结构 + Linux + 计网 + 面试 33～36）。LLMInfra / LLMPython 为 **扩展选学**。Java/Python FastAPI/AIAgent 为备选。

**推荐顺序（C++ 核心）**：

```text

  ↓

  ↓

  ↓

  ↓

```

## 2. 文件编码

- 所有 `.md` 文件必须使用 **UTF-8** 编码
- HTML 示例必须在 `<head>` 中包含 `<meta charset="UTF-8" />`
- 编辑时检查编辑器右下角编码显示

## 3. 文件命名规范

- 格式：`编号-主题关键词.md`
- 编号用两位数字，不足两位补 0（如 `06-`）
- 主题用中文，多个关键词用 `-` 连接
- 示例：`06-JavaScript基础语法与数据类型.md`、`01-TypeScript入门与环境配置.md`

## 4. 单篇文档内容规范

### 4.1 章节结构

- 每个大知识点作为 `##` 二级标题，从 1 开始编号
- 子知识点作为 `###` 三级标题
- 每份文档末尾必须有 **「练习建议」** 和 **「学完标准」**
- 框架/后端系列额外建议：**「本章衔接」**（开篇）+ **「下一章预告」**（章末）

### 4.2 七类必补内容（Vue / React / Java / TypeScript / AIAgent 强制执行）

| 类型 | 说明 | 最低要求 |
|------|------|----------|
| 手把手实操 | 从零跟做 | 每章 ≥1 段完整可跟做流程 |
| 常见报错表 | 现象 → 原因 → 解决 | ≥8 行 |
| 深入解释 | 「为什么」+ 小案例 | 核心概念 ≥2 处 |
| 命令预期输出 | 终端命令 + 成功时长什么样 | 涉及 CLI 的章节必须有 |
| 练习参考答案 | 分级练习 + 参考答案 | 基础题必给答案 |
| 章节衔接 | 与上/下章、平行系列对照 | 开篇 + 章末各 1 段 |
| Mermaid 图 | 流程/结构可视化 | 每章 ≥1 张（路线图除外） |

### 4.2.1 零基础精通扩充块（2026-06 起，AIAgent / Java / 计网 / Git / Linux / Web安全 等逐步对齐）

每章扩充时须增加（详见 [`.cursor.memory/EXPANSION-STANDARD.md`](.cursor.memory/EXPANSION-STANDARD.md)）：

| 块 | 说明 |
|----|------|
| §0 读前导读 | 0.1 一句话 … 0.5 可验证成果 |
| 知识地图 | checkbox 全覆盖，学完逐项 ☑ |
| 术语三件套 | 定义 + 生活类比 + 为什么重要 |
| 手把手步骤表 | 步骤 \| 动作 \| 预期 \| 若不对 |
| 逐行读 | 对 >10 行代码/JSON |
| FAQ | ≥10 条常见困惑 |
| 闭卷自测 | ≥10 题 + 参考答案 |
| 费曼检验 | 3 分钟口述提纲 |

事实核查：Spring AI 等见 `.cursor.memory/verified-facts/`；不确定则链官方文档，禁止瞎编 API。

### 4.3 代码示例规范

- 每个重要知识点应配有 **完整可运行** 的示例
- JS/TS 使用 2 空格缩进
- TypeScript 示例须标注 `// ts` 或 `lang="typescript"`，并说明如何运行（`tsc` / `vite` / `tsx`）
- HTML/CSS 示例尽量完整（含 DOCTYPE、charset、viewport）

### 4.4 术语规范

- 统一使用中文术语（如「变量」不用 variable）
- 首次出现的英文术语附中文解释
- 代码中的变量名使用英文驼峰命名

### 4.5 符号规范

- 中英文之间加空格（如：`使用 CSS 布局`）
- 代码块使用 ``` 包裹，标注语言类型
- 行内代码使用单个反引号

### 4.6 交叉链接规范

- 跨系列链接使用相对路径，路径须含子目录名：
  - 前端 → 后端：`../../后端学习/Java/xx.md` 或 `../../后端学习/Python/xx.md` 或 `../../后端学习/C++/xx.md` 或 `../../后端学习/数据结构/xx.md`
  - 后端 → 前端：`../../前端学习/Vue/xx.md`
  - 同层：`../Vue/xx.md`
- 移动文件后必须全局搜索旧路径并更新
